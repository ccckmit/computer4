# math4rs v0.7 規劃書：幾何模組 (geometry/)

## 1. 目標

實作幾何相關功能，包括點、向量、線、平面、圓、球、多邊形、距離計算和幾何變換，與 math4js 的 geometry 模組相容。

## 2. geometry/ 模組結構

```
src/geometry/
├── mod.rs              # 統一 export
├── point.rs           # 點 (Point2D, Point3D, PointND)
├── vector.rs          # 向量 (Vec2, Vec3, VecN)
├── line.rs            # 2D 直線 (Line2D)
├── plane.rs           # 平面 (Plane)
├── circle.rs          # 圓與弧 (Circle, Arc)
├── sphere.rs          # 球體 (Sphere)
├── polygon.rs         # 多邊形 (Polygon)
├── distance.rs        # 距離計算
└── transform.rs       # 2D/3D 幾何變換
```

## 3. API 設計（與 math4js 相容）

### 3.1 point.rs — 點

| API | 說明 |
|-----|------|
| `Point2D::new(x, y)` | 建立 2D 點 |
| `Point3D::new(x, y, z)` | 建立 3D 點 |
| `PointND::new(coords)` | 建立 N 維點 |
| `distance(&self, other)` | 計算兩點距離 |

### 3.2 vector.rs — 向量

| API | 說明 |
|-----|------|
| `Vec2::new(x, y)` | 建立 2D 向量 |
| `Vec3::new(x, y, z)` | 建立 3D 向量 |
| `VecN::new(coords)` | 建立 N 維向量 |
| `add, sub, mul` | 向量加減乘法 |
| `dot(&self, v)` | 點積 |
| `cross(&self, v)` | 叉積 (Vec2, Vec3) |
| `length()` | 向量長度 |
| `normalize()` | 單位化 |

### 3.3 line.rs — 2D 直線

| API | 說明 |
|-----|------|
| `Line2D::new(p1, p2)` | 建立直線 |
| `contains_point(&self, p)` | 點是否在直線上 |
| `distance_to(&self, p)` | 點到直線距離 |
| `parallel_to(&self, other)` | 兩直線是否平行 |
| `intersection_with(&self, other)` | 兩直線交點 |

### 3.4 plane.rs — 平面

| API | 說明 |
|-----|------|
| `Plane::new(normal, point)` | 建立平面 |
| `Plane::from_points(p1, p2, p3)` | 由三點建立平面 |
| `distance_to(&self, point)` | 點到平面距離 |
| `contains_point(&self, p)` | 點是否在平面上 |
| `intersect_line(&self, p1, p2)` | 直線與平面交點 |

### 3.5 circle.rs — 圓與弧

| API | 說明 |
|-----|------|
| `Circle::new(center, radius)` | 建立圓 |
| `area()` | 圓面積 |
| `circumference()` | 圓周長 |
| `contains_point(&self, p)` | 點是否在圓內 |
| `tangent(&self, p)` | 圓的切線 |
| `Arc::new(center, radius, start_angle, end_angle)` | 建立弧 |
| `length()` | 弧長 |
| `area()` | 弧形面積 |

### 3.6 sphere.rs — 球體

| API | 說明 |
|-----|------|
| `Sphere::new(center, radius)` | 建立球體 |
| `volume()` | 球體積 |
| `surface_area()` | 球表面積 |
| `contains_point(&self, p)` | 點是否在球內 |
| `intersect_line(&self, p1, p2)` | 直線與球交點 |

### 3.7 polygon.rs — 多邊形

| API | 說明 |
|-----|------|
| `Polygon::new(vertices)` | 建立多邊形 |
| `area()` | 多邊形面積 |
| `perimeter()` | 多邊形周長 |
| `contains_point(&self, p)` | 點是否在多邊形內 |
| `centroid()` | 重心 |
| `triangle_area(p1, p2, p3)` | 三角形面積 |
| `quadrilateral_area(p1, p2, p3, p4)` | 四邊形面積 |

### 3.8 distance.rs — 距離計算

| API | 說明 |
|-----|------|
| `point_to_point_2d(p1, p2)` | 2D 點到點距離 |
| `point_to_point_3d(p1, p2)` | 3D 點到點距離 |
| `point_to_line_2d(p, line)` | 點到直線距離 |
| `line_to_line_2d(l1, l2)` | 兩直線間距離 |

### 3.9 transform.rs — 幾何變換

| API | 說明 |
|-----|------|
| `Transform2D::identity()` | 單位變換 |
| `Transform2D::translation(x, y)` | 平移變換 |
| `Transform2D::scaling(sx, sy)` | 縮放變換 |
| `Transform2D::rotation(angle)` | 旋轉變換 |
| `apply(&self, p)` | 應用變換到點 |
| `compose(&self, other)` | 組合變換 |
| `Transform3D::identity()` | 3D 單位變換 |
| `Transform3D::translation(x, y, z)` | 3D 平移 |
| `Transform3D::scaling(sx, sy, sz)` | 3D 縮放 |
| `Transform3D::rotation_x/y/z(angle)` | 3D 旋轉 |
