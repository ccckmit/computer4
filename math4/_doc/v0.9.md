# math4rs v0.9 規劃書：數論模組 (number_theory/)

## 1. 目標

實作數論相關功能，包括質數、組合數學和模運算，與 math4js 的 number_theory 模組相容。

## 2. number_theory/ 模組結構

```
src/number_theory/
├── mod.rs              # 統一 export
├── primes.rs           # 質數相關函數
├── combinatorics.rs    # 組合數學（階乘、斐波那契、二項式）
└── modular.rs          # 模運算
```

## 3. API 設計（與 math4js 相容）

### 3.1 primes.rs — 質數

| API | 說明 |
|-----|------|
| `is_prime(n)` | 判斷是否為質數 |
| `primes_up_to(n)` | 回傳小於等於 n 的所有質數 |
| `prime_factors(n)` | 質因數分解 |
| `gcd(a, b)` | 最大公因數 |
| `lcm(a, b)` | 最小公倍數 |

### 3.2 combinatorics.rs — 組合數學

| API | 說明 |
|-----|------|
| `factorial(n)` | 階乘 n! |
| `fibonacci(n)` | 斐波那契數 F(n) |
| `fibonacci_sequence(n)` | 前 n 個斐波那契數 |
| `is_fibonacci(n)` | 判斷是否為斐波那契數 |
| `binomial(n, k)` | 二項式係數 C(n,k) |

### 3.3 modular.rs — 模運算

| API | 說明 |
|-----|------|
| `mod_pow(base, exp, mod_base)` | 模指數運算 |
| `mod_inv(a, m)` | 模反元素 |
| `extended_gcd(a, b)` | 擴展歐幾里得演算法 |
| `mod_add(a, b, m)` | 模加法 |
| `mod_sub(a, b, m)` | 模減法 |
| `mod_mul(a, b, m)` | 模乘法 |
| `chinese_remainder(remainders, moduli)` | 中國剩餘定理 |
| `is_coprime(a, b)` | 判斷是否互質 |
| `euler_totient(n)` | 歐拉 phi 函數 |
| `is_primitive_root(g, p)` | 判斷是否為原根 |
