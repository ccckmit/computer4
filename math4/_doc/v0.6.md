# math4rs v0.6 規劃書：微積分模組 (calculus/)

## 1. 目標

實作微積分相關功能，包括微分、積分、多變數微積分、數列級數、泰勒展開和數值優化，與 math4js 的 calculus 模組相容。

## 2. calculus/ 模組結構

```
src/calculus/
├── mod.rs              # 統一 export
├── derivative.rs       # 單變數微分
├── integral.rs        # 單變數積分
├── multivariable.rs   # 多變數微積分
├── sequence.rs        # 數列與級數
├── taylor.rs          # 泰勒展開
└── optimize.rs        # 數值優化
```

## 3. API 設計（與 math4js 相容）

### 3.1 derivative.rs — 單變數微分

| API | 說明 |
|-----|------|
| `derivative(f, h) -> Fn(f64) -> f64` | 數值導數（中央差分） |
| `second_derivative(f, h) -> Fn(f64) -> f64` | 二階導數 |
| `partial(f, i, h) -> Fn(&[f64]) -> f64` | 偏導數 |
| `gradient(f, h) -> Fn(&[f64]) -> Vec<f64>` | 梯度向量 |
| `jacobian(f, h) -> Fn(&[f64]) -> Vec<Vec<f64>>` | Jacobian 矩陣 |
| `directional_derivative(f, x, direction)` | 方向導數 |

### 3.2 integral.rs — 單變數積分

| API | 說明 |
|-----|------|
| `trapezoid(f, a, b, n)` | 梯形法 |
| `simpson(f, a, b, n)` | Simpson 法 |
| `romberg(f, a, b, tol, max_iter)` | Romberg 積分 |
| `adaptive(f, a, b, tol)` | 自適應積分 |
| `monte_carlo(f, a, b, n)` | Monte Carlo 積分 |
| `gauss_legendre(f, a, b, n)` | Gauss-Legendre 積分 |

### 3.3 multivariable.rs — 多變數微積分

| API | 說明 |
|-----|------|
| `grad(f, h) -> Fn(&[f64]) -> Vec<f64>` | 梯度 |
| `divergence(F, h) -> Fn(&[f64]) -> f64` | 散度 |
| `curl_2d(F) -> Fn(&[f64]) -> f64` | 2D 旋度 |
| `curl_3d(F) -> Fn(&[f64]) -> Vec<f64>` | 3D 旋度 |
| `laplacian(f, h) -> Fn(&[f64]) -> f64` | 拉普拉斯算子 |
| `hessian(f, h) -> Fn(&[f64]) -> Vec<Vec<f64>>` | Hessian 矩陣 |

### 3.4 sequence.rs — 數列與級數

| API | 說明 |
|-----|------|
| `sequence(f, n0, n) -> Vec<f64>` | 產生數列 |
| `series(f, a, b) -> f64` | 級數和 |
| `converge(f, x0, tol, max_iter) -> ConvergeResult` | 定點收斂 |
| `limit(f, a, direction, tol) -> Option<f64>` | 極限 |
| `infinite_series(f, tol, max_terms) -> SeriesResult` | 無窮級數 |
| `alternating_series(f, tol, max_terms) -> SeriesResult` | 交錯級數 |

### 3.5 taylor.rs — 泰勒展開

| API | 說明 |
|-----|------|
| `taylor(f, a, n, h) -> Fn(f64) -> f64` | Taylor 級數 |
| `maclaurin(f, n, h) -> Fn(f64) -> f64` | Maclaurin 級數 |
| `series_sum(f, a, n) -> f64` | 級數求和 |
| `power_series_coeffs(f, n, center, h) -> Vec<f64>` | 冪級數係數 |

### 3.6 optimize.rs — 數值優化

| API | 說明 |
|-----|------|
| `golden_section(f, a, b, tol, max_iter) -> f64` | 黃金分割搜尋 |
| `gradient_descent(f, df, x0, lr, tol, max_iter) -> OptimizeResult` | 梯度下降法 |
| `newton_method(f, df, ddf, x0, tol, max_iter) -> NewtonResult` | Newton 法 |
| `conjugate_gradient(A, b, x0, tol, max_iter) -> OptimizeResult` | 共軛梯度法 |
| `momentum_gradient_descent(f, df, x0, lr, momentum, tol, max_iter) -> OptimizeResult` | 動量梯度下降 |
