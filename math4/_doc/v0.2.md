# math4rs v0.2 規劃書：繪圖模組 (plot/)

## 1. 目標

使用 [plotters](https://github.com/plotters-rs/plotters) 套件包裝一個 R-style 的繪圖 API，與 math4js 的 `rplot.ts` 保持相同的使用體驗。

## 2. Cargo.toml 新增依賴

```toml
[dependencies]
plotters = "0.3"
plotters-bitmap = "0.1"
image = "0.24"
```

## 3. plot/ 模組結構

```
src/plot/
├── mod.rs              # 統一 export
├── rplot.rs            # R-style 繪圖 API
├── device.rs           # 輸出設備管理 (PNG/PDF)
└── options.rs          # 繪圖選項
```

## 4. API 設計（與 math4js 相容）

### 4.1 設備函數 (Device Functions)

| API | 說明 |
|-----|------|
| `png(filename: &str, width: u32, height: u32)` | 開啟 PNG 輸出裝置 |
| `svg(filename: &str, width: u32, height: u32)` | 開啟 SVG 輸出 |
| `dev_off()` | 關閉並儲存圖形 |
| `clear()` | 清除當前圖形 |

### 4.2 繪圖函數 (Plotting Functions)

#### plot — 散佈圖 / 線圖 / 兩者

```rust
pub fn plot(x: &[f64], y: &[f64], ptype: PlotType, options: PlotOptions) -> Result<(), Box<dyn Error>>
```

| 參數 | 說明 |
|------|------|
| `ptype` | `PlotType::Points`, `PlotType::Line`, `PlotType::Both` |
| `options.col` | 顏色 |
| `options.ms` | 點大小 |
| `options.lwd` | 線寬 |
| `options.name` | 圖例名稱 |
| `options.main` | 標題 |
| `options.xlab` | X 軸標籤 |
| `options.ylab` | Y 軸標籤 |
| `options.xlim` | X 軸範圍 |
| `options.ylim` | Y 軸範圍 |
| `options.showlegend` | 顯示圖例 |

#### hist — 直方圖

```rust
pub fn hist(x: &[f64], options: HistOptions) -> Result<(), Box<dyn Error>>
```

| 參數 | 說明 |
|------|------|
| `options.breaks` | 分箱數量 |
| `options.col` | 顏色 |
| `options.border` | 邊框顏色 |
| `options.freq` | true=頻率, false=機率密度 |

#### boxplot — 箱形圖

```rust
pub fn boxplot(data: &[&[f64]], options: BoxPlotOptions) -> Result<(), Box<dyn Error>>
```

| 參數 | 說明 |
|------|------|
| `options.col` | 顏色 |
| `options.names` | 組別名稱 |
| `options.showpoints` | 是否顯示點 |

#### qqnorm — Q-Q 圖

```rust
pub fn qqnorm(x: &[f64], options: PlotOptions) -> Result<(), Box<dyn Error>>
```

## 5. 實作重點

### 5.1 全域狀態管理

使用 `std::cell::RefCell` 管理當前繪圖設備的狀態：

```rust
thread_local! {
    static CURRENT_DEVICE: RefCell<Option<PlotDevice>> = RefCell::new(None);
}
```

### 5.2 輸出格式支援

| 格式 | 方法 |
|------|------|
| PNG | `plotters-bitmap` + `image` |
| SVG | `plotters` 內建 SVG backend |

### 5.3 顏色名稱對照

與 R 的 `col` 參數相容：

```rust
fn parse_color(col: &str) -> RGBColor
```

支援：`"red"`, `"blue"`, `"green"`, `"black"`, `"#FF0000"` 等。
