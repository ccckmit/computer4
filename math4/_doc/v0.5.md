# math4rs v0.5 規劃書：代數模組 (algebra/)

## 1. 目標

實作代數相關功能，包括多項式、複數、有理數和求根演算法，與 math4js 的 algebra 模組相容。

## 2. algebra/ 模組結構

```
src/algebra/
├── mod.rs              # 統一 export
├── polynomial.rs       # 多項式類別
├── complex.rs          # 複數類別
├── rational.rs         # 有理數類別
└── roots.rs            # 求根演算法
```

## 3. API 設計（與 math4js 相容）

### 3.1 polynomial.rs — 多項式類別

```rust
pub struct Polynomial { coeffs: Vec<f64> }
```

| API | 說明 |
|-----|------|
| `Polynomial::new(coeffs: Vec<f64>) -> Polynomial` | 建構多項式 |
| `p.coeffs() -> Vec<f64>` | 取得係數陣列 |
| `p.degree() -> i32` | 多項式次數 |
| `p.eval(x: f64) -> f64` | 代入 x 求值 |
| `p.add(&other) -> Polynomial` | 多項式加法 |
| `p.sub(&other) -> Polynomial` | 多項式減法 |
| `p.mul(&other) -> Polynomial` | 多項式乘法 |
| `p.scalar_mul(s) -> Polynomial` | 純量乘法 |
| `p.derivative() -> Polynomial` | 導函數 |
| `p.integral(c) -> Polynomial` | 不定積分 |
| `p.compose(&other) -> Polynomial` | 合成函數 |
| `p.divide(&d) -> (quotient, remainder)` | 多項式除法 |
| `p.mod(&d) -> Polynomial` | 取餘數 |
| `p.gcd(&other) -> Polynomial` | 最大公因式 |
| `p.equals(&other) -> bool` | 判斷相等 |

**注意**：係數陣列為遞增順序 `[a0, a1, a2, ...]` 表示 `a0 + a1*x + a2*x^2 + ...`

### 3.2 complex.rs — 複數類別

```rust
pub struct Complex { re: f64, im: f64 }
```

| API | 說明 |
|-----|------|
| `Complex::new(re, im) -> Complex` | 建構複數 |
| `c.re() -> f64` | 取得實部 |
| `c.im() -> f64` | 取得虛部 |
| `c.add(&other) -> Complex` | 複數加法 |
| `c.sub(&other) -> Complex` | 複數減法 |
| `c.mul(&other) -> Complex` | 複數乘法 |
| `c.div(&other) -> Complex` | 複數除法 |
| `c.conj() -> Complex` | 共軛複數 |
| `c.abs() -> f64` | 模（絕對值） |
| `c.arg() -> f64` | 幅角 |
| `c.sqrt() -> Complex` | 平方根 |
| `c.exp() -> Complex` | 指數 |
| `c.log() -> Complex` | 對數 |
| `c.pow(n) -> Complex` | 冪運算 |
| `c.sin() -> Complex` | 正弦 |
| `c.cos() -> Complex` | 餘弦 |
| `c.tan() -> Complex` | 正切 |
| `c.to_polar() -> (r, theta)` | 極座標 |
| `Complex::from_polar(r, theta) -> Complex` | 從極座標建立 |
| `Complex::i() -> Complex` | 虛數單位 i |

### 3.3 rational.rs — 有理數類別

```rust
pub struct Rational { num: i64, den: i64 }
```

| API | 說明 |
|-----|------|
| `Rational::new(num, den) -> Rational` | 建構有理數（自動約分） |
| `r.num() -> i64` | 取得分子 |
| `r.den() -> i64` | 取得分母 |
| `r.add(&other) -> Rational` | 加法 |
| `r.sub(&other) -> Rational` | 減法 |
| `r.mul(&other) -> Rational` | 乘法 |
| `r.div(&other) -> Rational` | 除法 |
| `r.simplify() -> Rational` | 化簡 |
| `r.to_number() -> f64` | 轉換為浮點數 |
| `r.equals(&other) -> bool` | 判斷相等 |
| `parse_rational(s: &str) -> Rational` | 解析字串 |

### 3.4 roots.rs — 求根演算法

| API | 說明 |
|-----|------|
| `bisection(f, a, b, tol, max_iter)` | 二分法 |
| `newton(f, df, x0, tol, max_iter)` | 牛頓法 |
| `secant(f, x0, x1, tol, max_iter)` | 割線法 |
| `horner(coeffs, x)` | Horner 快速求值 |
| `horner_derivative(coeffs, x)` | Horner 導數求值 |
| `find_all_roots(f, coeffs, tol, max_iter)` | 找所有根 |
| `deflation(coeffs, root)` | 多項式除去已知根 |
