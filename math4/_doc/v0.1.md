# math4rs v0.1 規劃書：統計模組 (statistics/)

## 1. 專案初始化

### 1.1 建立 Cargo 專案結構

```
math4rs/
├── Cargo.toml
├── src/
│   └── lib.rs          # 主 library 入口
├── src/statistics/     # 統計模組
├── tests/              # 整合測試
└── _doc/               # 文件
```

### 1.2 Cargo.toml 依賴

```toml
[package]
name = "math4rs"
version = "0.1.0"
edition = "2021"

[dependencies]
statrs = "0.16"
rand = "0.8"
```

## 2. statistics/ 模組結構

```
src/statistics/
├── mod.rs              # 統一 export
├── stats.rs            # 敘述統計
├── distributions.rs    # 機率分布
├── random.rs           # 亂數產生器
├── interval.rs         # 區間估計
├── hypothesis.rs       # 假設檢定
├── theorem.rs          # 統計定理驗證
└── info.rs             # 資訊理論
```

## 3. 各子模組 API 設計

### 3.1 stats.rs — 敘述統計函數

| API | 說明 |
|-----|------|
| `mean(x: &[f64]) -> f64` | 平均數 |
| `median(x: &[f64]) -> f64` | 中位數 |
| `variance(x: &[f64], ddof: usize) -> f64` | 變異數 |
| `sd(x: &[f64], ddof: usize) -> f64` | 標準差 |
| `covariance(x: &[f64], y: &[f64], ddof: usize) -> f64` | 共變異數 |
| `correlation(x: &[f64], y: &[f64]) -> f64` | 相關係數 |
| `quantile(x: &[f64], p: f64) -> f64` | 分位數 |
| `iqr(x: &[f64]) -> f64` | 四分位距 |
| `z_score(x: f64, mu: f64, sigma: f64) -> f64` | Z 分數 |
| `standardize(x: &[f64]) -> Vec<f64>` | 標準化 |
| `summary(x: &[f64]) -> Summary` | 摘要統計 |
| `min_val / max_val / range_stat / sum_stat` | 基本統計 |
| `mode(x: &[f64]) -> Vec<f64>` | 眾數 |
| `skewness(x: &[f64]) -> f64` | 偏度 |
| `kurtosis(x: &[f64]) -> f64` | 峰度 |

### 3.2 distributions.rs — 機率分布

實作與 math4js 相同的 API 命名：

| API | 說明 |
|-----|------|
| `dnorm / pnorm / qnorm / rnorm` | 常態分布 |
| `dt / pt / qt / rt` | Student t 分布 |
| `dchisq / pchisq / qchisq / rchisq` | 卡方分布 |
| `df / pf / qf / rf` | F 分布 |
| `dbinom / pbinom / qbinom / rbinom` | 二項分布 |
| `dpois / ppois / qpois / rpois` | 卜瓦松分布 |

### 3.3 random.rs — 亂數產生器

基於 LCG (線性同餘生成器)：

| API | 說明 |
|-----|------|
| `set_seed(seed: u64)` | 設定種子 |
| `get_seed() -> u64` | 取得種子 |
| `reset_seed()` | 重置種子 |
| `random() -> f64` | 均勻分布 [0, 1) |
| `rand_int(min, max) -> i64` | 整數乱數 |
| `random_batch(n) -> Vec<f64>` | 批量生成 |
| `rand_int_batch(n, min, max) -> Vec<i64>` | 批量整數生成 |
| `set_seed_string(s: &str)` | 字串設定種子 |
| `mix(a, b) -> u64` | 混合種子 |

### 3.4 interval.rs — 區間估計

| API | 說明 |
|-----|------|
| `conf_interval(x, alpha) -> ConfIntervalResult` | t 分佈信賴區間 |
| `conf_interval_proportion(x, n, alpha)` | 比例信賴區間 |
| `bootstrap_ci(x, statistic_fn, n_bootstrap, alpha)` | Bootstrap CI |
| `conf_interval_variance(x, alpha)` | 變異數信賴區間 |

### 3.5 hypothesis.rs — 假設檢定

| API | 說明 |
|-----|------|
| `z_test(x, mu0, sigma, alpha)` | Z 檢定 |
| `t_test(x, mu0, alpha)` | 單一樣本 t 檢定 |
| `t_test_two(x1, x2, alpha)` | 獨立雙樣本 t 檢定 |
| `t_test_paired(x1, x2, alpha)` | 配對 t 檢定 |
| `chisq_test(observed, expected, alpha)` | 卡方檢定 |
| `anova(groups, alpha)` | 單因子變異數分析 |

### 3.6 theorem.rs — 統計定理驗證

| API | 說明 |
|-----|------|
| `central_limit_theorem(...)` | 中央極限定理驗證 |
| `law_of_large_numbers(...)` | 大數法則驗證 |
| `chebyshev_inequality(...)` | 柴比斯夫不等式 |
| `chebyshev_verify(...)` | 柴比斯夫不等式驗證 |
| `markov_inequality(...)` | 馬可夫不等式 |
| `markov_verify(...)` | 馬可夫不等式驗證 |
| `bernoulli_verify(...)` | 白努利分布驗證 |
| `bayes_theorem(...)` | 貝氏定理 |
| `bayes_verify(...)` | 貝氏定理驗證 |
| `information_entropy(...)` | 資訊熵 |
| `information_entropy_verify(...)` | 資訊熵驗證 |
| `mutual_information(...)` | 互資訊 |

### 3.7 info.rs — 資訊理論

| API | 說明 |
|-----|------|
| `entropy(p, base)` | 熵 |
| `cross_entropy(p, q, base)` | 交叉熵 |
| `kl_divergence(p, q, base)` | KL 散度 |
| `mutual_information(x, y, base)` | 互資訊 |
| `conditional_entropy(x, y, base)` | 條件熵 |
| `pmi(x, y, base)` | 點互資訊 |
