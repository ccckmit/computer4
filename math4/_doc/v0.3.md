# math4rs v0.3 規劃書：ndarray 模組

## 1. 目標

使用 Rust [ndarray](https://github.com/rust-ndarray/ndarray) 套件實作 NumPy 風格的多維陣列 API，與 math4js 的 ndarray 模組相容。

## 2. Cargo.toml 新增依賴

```toml
[dependencies]
ndarray = "0.15"
```

## 3. ndarray/ 模組結構

```
src/ndarray/
├── mod.rs          # 統一 export
├── core.rs         # ndarray 結構、工廠函數
├── operations.rs   # element-wise 運算、聚合、比較
└── random.rs       # 亂數產生
```

## 4. API 設計（與 math4js 相容）

### 4.1 core.rs — 核心結構與工廠

| API | 說明 |
|-----|------|
| `array(data: &[f64], shape: &[usize]) -> ArrayD<f64>` | 從扁平資料建立任意維度陣列 |
| `zeros(shape: &[usize]) -> ArrayD<f64>` | 零矩陣 |
| `ones(shape: &[usize]) -> ArrayD<f64>` | 全 1 矩陣 |
| `full(shape: &[usize], fill: f64) -> ArrayD<f64>` | 填滿固定值 |
| `eye(n: usize) -> Array2<f64>` | 單位矩陣 |
| `identity(n: usize) -> Array2<f64>` | 單位矩陣（同 eye） |
| `diag(v: &[f64], k: isize) -> Array2<f64>` | 對角矩陣 |
| `arange(start: f64, stop: f64, step: f64) -> Array1<f64>` | 範圍序列 |
| `linspace(start: f64, stop: f64, num: usize) -> Array1<f64>` | 等差序列 |
| `concatenate(arrays: &[ArrayD<f64>], axis: Axis) -> ArrayD<f64>` | 串接 |
| `vstack(arrays: &[ArrayD<f64>]) -> ArrayD<f64>` | 垂直堆疊 |
| `hstack(arrays: &[ArrayD<f64>]) -> ArrayD<f64>` | 水平堆疊 |

| API | 說明 |
|-----|------|
| `add(a: Array, b: Array) -> Array` | 加法 |
| `sub(a: Array, b: Array) -> Array` | 減法 |

| `div(a: Array, b: Array) -> Array` | 除法 |
| `matmul(a: Array, b: Array) -> Array` | 矩陣乘法 |
| `dot(a: Array, b: Array) -> f64` | 點積 / 矩陣積 |
| `abs(a: Array) -> Array` | 絕對值 |
| `sqrt(a: Array) -> Array` | 開根號 |
| `exp(a: Array) -> Array` | 指數 |
| `log(a: Array) -> Array` | 對數 |
| `sin / cos / tan` | 三角函數 |

| `mean(a: &Array, axis: Option<usize>) -> Result<Array, f64>` | 平均 |
| `std(a: &Array, axis: Option<usize>) -> Result<Array, f64>` | 標準差 |

| `gt / gte / lt / lte / eq / ne` | 比較運算 |

### 4.3 random.rs — 亂數

| API | 說明 |
|-----|------|
| `rand(shape: &[usize]) -> Array` | 常態分布 U(0,1) |
| `randn(shape: &[usize]) -> Array` | 標準常態分布 |
| `randint(low: usize, high: usize, shape: &[usize]) -> Array` | 整數均勻分布 |
| `choice(a: &Array, n: usize, replace: bool) -> Array` | 抽樣 |
| `shuffle(a: &mut Array)` | 原地洗牌 |

## 5. 實作重點

### 5.1 ndarray 封裝

使用 `ArrayD<f64>` 為核心，配合 `Array2<f64>` 做矩陣特化：

```rust
pub type Array = ArrayD<f64>;
pub type Matrix = Array2<f64>;
pub type Vector = Array1<f64>;
```

### 5.2 Axis 處理

使用 `ndarray::Axis`，aggregation 函式回傳 `Result<Array, f64>`。
