# v0.16 — 包裝工具 / IPC / ELF 工具

## 新增工具

### 極簡單（7 個）

| 工具 | 說明 |
|------|------|
| `users` | 列出登入使用者名稱（空格分隔） |
| `sum` | 舊式校驗和（SysV 32-bit / BSD 16-bit） |
| `mktemp` | 建立暫存檔案或目錄 |
| `c99` | C 編譯器包裝（呼叫 `cc`） |
| `fort77` | FORTRAN 編譯器包裝（呼叫 `f77`） |
| `lex` | 詞法分析器包裝（呼叫 `lex`） |
| `yacc` | 語法分析器包裝（呼叫 `yacc`） |

### 中等（5 個）

| 工具 | 說明 |
|------|------|
| `ar` | BSD ar 歸檔格式建立/列表/解壓縮/取代 |
| `ipcrm` | 移除 SysV IPC（共享記憶體、號誌） |
| `ipcs` | 顯示 SysV IPC 狀態 |
| `strip` | 移除 ELF 符號表（`.symtab`/`.strtab`） |
| `nm` | 列出 ELF 動態符號 |

## 實作細節

- `ar` 實作 BSD ar 格式（`#1/<len>` 長名稱），支援 `-t`/`-x`/`-r`/`-c`/`-v`
- `nm` 解析 ELF32/ELF64，列出全域符號（`STB_GLOBAL`），輸出格式 `value type name`
- `strip` 清除 ELF 的 `.symtab` 和 `.strtab` 節區及對應節區表項
- `ipcs` 使用 `shmctl(IPC_STAT)` 掃描共享記憶體段（前 256 個 id）
- `mktemp` 在 `/tmp/` 下產生唯一名稱，支援 `-d`（目錄）和自訂樣板

## 整合測試

13 個新測試案例：users（執行正常）、sum（SysV/BSD 格式）、mktemp（檔案/目錄/樣板）、ar（建立/列表/解壓縮）、ipcrm/ipcs（基本執行）、strip/nm（非 ELF 拒絕）
