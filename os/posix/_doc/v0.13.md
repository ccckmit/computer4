# v0.13 — 大型工具

## 新增工具

| 工具 | POSIX | 說明 | 關鍵實作 |
|------|-------|------|----------|
| `ed` | 強制 | 行編輯器 | 互動 REPL：`a`/`i`/`c` 插入、`d` 刪除、`p`/`n` 顯示、`s` 取代、`w` 寫入、`q` 離開、`=` 行數 |
| `awk` | 強制 | 樣式掃描語言 | `{ print }`、`{ print $1 }`、`BEGIN`/`END` 區塊、樣式比對（`/pattern/` + 空=全部） |
| `bc` | 強制 | 任意精確度計算機 | REPL：四則運算 + `^`、變數、`scale`、`ibase`/`obase`（10/16/8/2） |
| `m4` | 強制 | 巨集處理器 | `define`/`undefine`、`ifdef`、`include`、`divert`/`undivert`、`eval`、`len`、`substr` |
| `pax` | 強制 | 可攜式歸檔交換 | `-w` 寫入 tar 格式、`-r` 列表、`-x` 解壓縮、`-f` 指定檔案 |

## 實作細節

- `ed` 使用簡單 REPL 迴圈；支援 `,`/`%` 全範圍、`1,5p` 範圍列印、`+`/`-` 游標移動、`.` 結束插入模式
- `awk` 遞迴下降解析器支援 pattern/action pairs；`BEGIN`/`END` 特殊區塊；`$0`/`$1`...`$NF` 欄位變數；`/pattern/` 字串包含比對
- `bc` 使用調度場演算法（shunting-yard）解析中序表達式；支援變數賦值與 i64 整數算術
- `m4` 字元級解析器；處理內建巨集並展開自由文字中的定義；不支援自訂引號字元
- `pax` 實作 POSIX tar 格式（512-byte header、八進位大小/時間/權限、checksum）；支援目錄遞迴

## 整合測試

8 個新測試案例：awk（print/field/BEGIN）、bc（加法/乘法）、m4（define/expand）、pax（write archive + list）、ed（存在性驗證）
