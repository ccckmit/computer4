# v0.10 — 文字合併/分割工具

## 新增工具

| 工具 | POSIX | 說明 | 關鍵實作 |
|------|-------|------|----------|
| `join` | 強制 | 依共同欄位合併檔案 | `-t` 分隔字元、`-1`/`-2` 欄位指定、`split_whitespace()` 預設 |
| `paste` | 強制 | 合併檔案行 | `-d` 分隔字元、`-s` 序列模式、多檔案合併 |
| `split` | 強制 | 分割檔案 | `-l` 行數（含 `k` 後綴）、`-a` 後綴長度、`-d` 前綴 |
| `strings` | 強制 | 尋找可列印字串 | `-n` 最小長度（預設 4）、二進位掃描 |
| `cksum` | 強制 | CRC 校驗和 | 自製 CRC-32 實作（無外部依賴） |
| `tsort` | 強制 | 拓撲排序 | Kahn 演算法（BFS）、`HashMap` 鄰接表 |

## 實作細節

- `join` 預設以空白分割（`split_whitespace`）；`-t` 指定單一字元分隔符
- `paste` 平行合併多檔案（預設 `\t` 分隔）；`-s` 將多檔案串列合併
- `split` 預設 1000 行/檔案，輸出 `x00`、`x01`、…；支援 `-a` 後綴長度
- `strings` 僅擷取 4+ 連續可列印 ASCII 字元（含空白與 tab）
- `cksum` 實作標準 CRC-32（`0xEDB88320` 多項式），輸出格式 `CRC 大小 檔名`
- `tsort` 實作 Kahn 演算法，每對輸入指定相依性

## 整合測試

6 個新測試案例：join（雙檔案合併）、paste（平行合併）、split（依 3 行分割成多檔）、strings（二進位字串擷取）、cksum（CRC + 大小驗證）、tsort（相依性排序）
