# v0.11 — Shell 輔助工具

## 新增工具

| 工具 | POSIX | 說明 | 關鍵實作 |
|------|-------|------|----------|
| `time` | 強制 | 計時指令執行 | `Command::spawn()` + `Instant::now()` 計時 real time |
| `renice` | 強制 | 修改行程優先權 | `libc::setpriority(PRIO_PROCESS, ...)` + `getpriority` |
| `wait` | 強制 | 等待行程完成 | `libc::waitpid()` |
| `umask` | 強制 | 設定/顯示檔案模式遮罩 | `libc::umask()` 八進位顯示 + 設定 |
| `hash` | 強制 | 記住/查詢工具位置 | `PATH` 搜尋（shell 內建行為） |
| `type` | XSI | 顯示指令類型 | 區分 shell 內建與外部指令 |
| `command` | 強制 | 執行簡單指令 | `-v`/`-V` 查詢、`-p` 略過函數搜尋、`Command::spawn` |
| `alias` | 強制 | 定義/顯示別名 | 內建別名（`ls`, `grep`, `..`） |
| `unalias` | 強制 | 移除別名 | `-a` 全部移除 |

## 實作細節

- `time` 使用 `std::time::Instant` 計算耗時，輸出 real 時間（POSIX 要求 real/user/sys，目前僅 real）
- `renice` 直接修改指定 PID 的調度優先權，需相應權限
- `umask` 無引數時顯示目前遮罩（八進位）；接受八進位數字或符號模式
- `hash` 在 `PATH` 中搜尋可執行檔；無引數時列出所有已知 shell 內建
- `type` 檢查內建清單，然後搜尋 `PATH`
- `command` 繞過 shell 函數直接執行外部指令；`-v` 僅顯示路徑
- `alias`/`unalias` 作為獨立工具，別名不跨呼叫持久化

## 整合測試

9 個新測試案例：time（成功/失敗）、umask（輸出格式）、type（內建偵測）、command（true/false 退出碼）、alias（預設別名）、hash（PATH 搜尋）、renice（錯誤處理）
