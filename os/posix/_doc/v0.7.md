# v0.7 — Phase 7: Advanced Tools

## Tools implemented

| Tool | Options | Description |
|------|---------|-------------|
| `find` | `-name` `-type` | Find files in directory hierarchy |
| `tar` | `-c` `-x` `-f` | Tape archiver (tar format) |
| `patch` | (none) | Apply diff to file |
| `comm` | (none) | Compare sorted files line by line |
| `fold` | `-w W` | Fold/wrap lines at width |
| `fmt` | `-w W` | Simple paragraph formatter |
| `nl` | (none) | Number lines of files |
| `expand` | `-t N` | Convert tabs to spaces |
| `unexpand` | `-t N` | Convert spaces to tabs |

## Design decisions

- `find` does recursive walk; `-name` does substring matching, `-type` filters `f`/`d`/`l`
- `tar` reads/writes basic POSIX tar format (name + size header, no USTAR extensions)
- `patch` reads unified diff format and applies -/+ hunks
- `comm` outputs three-column format (file1-only, common, file2-only)
- `fold` wraps at column width, prefers breaking at spaces
- `fmt` collects blank-line-separated paragraphs and re-flows them
- `nl` numbers non-empty lines by default; `-b a` numbers all lines
- `expand` replaces tabs with spaces at tab stops; `-i` for initial-only
- `unexpand` converts runs of spaces back to tabs
