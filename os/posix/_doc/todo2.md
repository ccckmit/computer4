# POSIX Tools Remaining (v0.16+)

**已完成：100 個工具**。以下為 POSIX.1-2008 尚未實作工具，按實作複雜度分類。

## 極簡單（7 個）

| 工具 | 類別 | 說明 |
|------|------|------|
| `users` | 強制 | 列出登入使用者 |
| `sum` | 強制 | 舊式校驗和（BSD/SysV） |
| `mktemp` | 常見 | 建立暫存檔案/目錄 |
| `c99` | SD | C 編譯器包裝（呼叫 `cc`） |
| `fort77` | SD | FORTRAN 包裝（呼叫 `f77`） |
| `lex` | SD | 詞法分析器包裝（呼叫 `lex`） |
| `yacc` | SD | 語法分析器包裝（呼叫 `yacc`） |

## 中等（5 個）

| 工具 | 類別 | 說明 |
|------|------|------|
| `ar` | 強制 | 歸檔格式管理員 |
| `ipcrm` | 強制 | 移除 SysV IPC |
| `ipcs` | 強制 | 顯示 SysV IPC |
| `strip` | SD | 移除 ELF 符號 |
| `nm` | SD | 列出 ELF 符號 |

## 大型（暫緩 — 不實作）

| 工具 | 類別 | 說明 |
|------|------|------|
| `man` | 強制 | 線上手冊 — 需要外部格式化程式或內建 nroff |
| `make` | SD | 建置工具 — 完整實作很複雜 |
| `vi` / `ex` | UP | 互動編輯器 — 數萬行程式碼 |
| `lp` | 強制 | 列印系統 — 需要背景常駐程式 |
| `mailx` | 強制 | 郵件客戶端 — MBOX/SMTP 協定 |
| `at` / `batch` / `crontab` | 強制 | 排程 — 需要常駐行程 |
| `bg` / `fg` / `jobs` | UP | 工作控制（shell 內建） |
| `fc` | UP | 歷史指令 |
