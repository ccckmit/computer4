# v0.14 — 基礎建設與選用工具

## 新增工具

| 工具 | POSIX | 說明 | 關鍵實作 |
|------|-------|------|----------|
| `who` | XSI | 顯示登入使用者 | `getutxent()` 掃描 utmpx 記錄 |
| `cal` | XSI | 顯示月曆 | 計算星期與閏年、格式化月曆輸出 |
| `csplit` | 強制 | 上下文分割檔案 | `/pattern/` 模式分割、數字行數分割 |
| `getconf` | 強制 | 系統組態值 | `sysconf()` 查詢 + POSIX 強制最小值 |
| `iconv` | 強制 | 字符集轉換 | UTF-8 ↔ ASCII ↔ Latin1 轉換 |
| `compress` | XSI | LZW 壓縮 | LZW 壓縮演算法（12-bit 碼表） |
| `uncompress` | XSI | LZW 解壓縮 | LZW 解壓縮，去除 `.Z` 後綴 |
| `zcat` | XSI | 解壓縮至 stdout | LZW 解壓縮，輸出至標準輸出 |

## 實作細節

- `who` 讀取 utmpx 記錄顯示使用者名稱、終端機、登入時間（macOS API）
- `cal` 實作 Zeller 公式計算星期、閏年判斷；支援年參數顯示全年
- `csplit` 支援 `/regex/` 樣式分割與數字行數分割；輸出 `xx00`、`xx01`、…
- `getconf` 回報 POSIX 強制最小值（`ARG_MAX`、`PATH_MAX` 等）+ 執行期 `sysconf` 值
- `iconv` 支援 UTF-8、ASCII、Latin1（ISO-8859-1）間的轉換
- `compress`/`uncompress`/`zcat` 實作基本 LZW 壓縮（12-bit、4096 字典項、無可變寬度編碼）
