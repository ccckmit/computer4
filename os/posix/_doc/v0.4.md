## Tools implemented

| Tool | Options | Description |
|------|---------|-------------|
| `grep` | `-i` `-v` `-c` `-n` `-l` | Pattern search in files |
| `sed` | (see script) | Stream editor ‚Äî `s/old/new/g` |
| `xargs` | (none) | Build and execute command from stdin |

## Design decisions

- `grep` uses simple substring matching (no regex); `-i` for case-insensitive, `-v` for invert, `-c` for count, `-l` for files-with-matches
- `sed` implements only the most basic `s///` substitute command with optional `g` flag
- `xargs` splits stdin on whitespace, runs command once per argument
