# v0.17 — 大型/簡易版工具

## 新增工具（10 個）

| 工具 | 類別 | 說明 |
|------|------|------|
| `vi` | UP | 簡易螢幕編輯器（原始終端機模式、插入/指令/普通模式） |
| `make` | SD | 簡易 Makefile 建置工具（目標/依賴/配方、變數展開） |
| `man` | 強制 | 線上手冊檢視器（搜尋 MANPATH、mandoc/nroff 格式化） |
| `crontab` | 強制 | 排程表管理（-l/-e/-r/file） |
| `at` | 強制 | 排程單次工作（-l/-r/時間規格） |
| `batch` | 強制 | 低負載排程工作 |
| `bg` | UP | 背景工作控制（SIGCONT 傳送） |
| `fg` | UP | 前景工作控制（SIGCONT + wait + tcsetpgrp） |
| `jobs` | UP | 工作列表（讀取 /tmp/jobs_${USER}） |

## 實作細節

- **vi**：原始終端機模式（libc `tcsetattr`），普通模式（hjkl/0$w/b/G/dd/x/:/i/a/o），插入模式（文字輸入/Enter/Backspace），指令模式（:w/:q/:wq/:q!），`$EDITOR` 相容
- **make**：解析 Makefile 目標:依賴 + Tab 開頭配方，支援 `$(VAR)` 展開、`$@`/`$^` 自動變數、`.PHONY` 類比、檔案時間戳判斷是否重建
- **man**：搜尋 `MANPATH`（`man<N>/<name>.<N>`），依序嘗試 mandoc/nroff/groff -mandoc 格式化，`$PAGER` 分頁顯示
- **crontab**：儲存於 `~/.cron/<user>`，支援 `-l`（列表）、`-e`（$EDITOR 編輯）、`-r`（移除）、`file`（安裝）
- **at**：儲存於 `~/.at/`，-l 列出、-r 移除、`now` 立即、其他時間規格
- **bg/fg/jobs**：透過 `/tmp/jobs_${USER}` 檔案與 sh 整合，`bg <pid>` 送 SIGCONT，`fg <pid>` 送 SIGCONT + `tcsetpgrp` + `waitpid`

## 整合測試
