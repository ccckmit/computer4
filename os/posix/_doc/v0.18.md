## Implementation Details

### lp
- Positional file args (or stdin via `-`) — writes content to stdout when no `-d printer` given
- `-d printer` spools files to `~/.lp/<printer>/` with per-job files named `job_<timestamp>`
- `-n copies` replicates content into spool header
- Prints job ID (`<printer>-<timestamp>`) to stderr per POSIX

### mailx
- No-arg mode: reads `~/mbox`, lists messages with `From` / `Subject` headers (numbered)
- `-s subject recipient`: reads stdin body, appends RFC 2822-like message to `~/mail/sent` and `~/mbox`
- `-f file`: dumps raw mailbox content
- Numeric arg: displays specific message from `~/mbox` (split on `\nFrom `)
- No background daemon — direct file-based delivery
