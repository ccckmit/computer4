# v0.1 — 基礎工具集

## 建置結構

- Cargo workspace：`libposix`（共用函式庫）+ `tools`（16 支工具）
- 跨平臺設計：`std`（Linux/macOS/Windows MSYS2） + `cfg(target_os = "none")`（xv8 框架預留）
- 零外部相依（除 `libc` 用於 `uname`/`kill`/`id` 的系統呼叫）

## libposix 共用函式庫

| 模組 | 功能 |
|------|------|
| `io` | `Read`/`Write` trait、`Stdin`/`Stdout`/`Stderr` 包裝、跨平臺 sys 實作 |
| `fmt` | `mode_string()` 將 `st_mode` 轉 `-rwxr-xr-x` 字串；`format_number()` |
| `opt` | `OptParser` — POSIX getopt 風格選項解析（短選項分組、長選項、`--` 終止） |

## 工具清單

| 工具 | POSIX 選項 | 說明 |
|------|-----------|------|
| `echo` | `-n` | 輸出引數，空白分隔 |
| `true` | — | 回傳 exit 0 |
| `false` | — | 回傳 exit 1 |
| `yes` | — | 重複輸出 `y` 或自訂字串 |
| `cat` | `-u` | 讀取檔案串接至 stdout，支援 `-` 表示 stdin |
| `wc` | `-l`, `-w`, `-c`, `-m` | 計行/字/字元/位元組 |
| `basename` | — | 去掉目錄前綴，可選移除 suffix |
| `dirname` | — | 取出目錄部分 |
| `sleep` | — | 延遲指定秒數（支援小數） |
| `kill` | `-l`, `-s` | 傳送信號（支援名稱或編號） |
| `uname` | `-a`, `-s`, `-n`, `-r`, `-v`, `-m` | 系統資訊（透過 `libc::uname`） |
| `printenv` | — | 印環境變數值 |
| `env` | `-i`, `-u` | 修改環境後執行程式 |
| `whoami` | — | 顯示當前使用者名稱 |

| `hostname` | — | 顯示主機名稱 |

## 測試

- **libposix 單元測試**：10 項（`fmt::mode_string`、`opt` 解析器）
- **工具整合測試**：41 項（每支工具的基本行為 + POSIX 選項驗證）
- **全部 51 測試通過**，`cargo build` 零警告
- 支援 `cargo build --release` 最佳化編譯

## 已知限制

- `id` 的預設輸出僅印 `uid=..(..) gid=..(..)`，尚未實作 supplementary groups 完整輸出
- `kill` 在非 Unix 平臺無實作（回傳錯誤）
- xv8 `no_std` 後端為框架預留，尚未完整實作
- 部分工具（`touch`、`chmod`）的檔案時間/權限操作依賴 `libc`，尚未封裝進 `libposix`
