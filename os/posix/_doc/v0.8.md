# v0.8 — 小型輸出工具

## 新增工具

| 工具 | 說明 | 關鍵實作 |
|------|------|----------|
| `printf` | 格式化輸出 | `%s`/`%d`/`%x`/`%f`/`%e`/`%g`、`%b`、`\\n`/`\\t` 跳脫、`*` 寬度 |
| `expr` | 整數表達式求值 | `+` `-` `*` `/` `%` `=` `!=` `>` `>=` `<` `<=` `|` `&`、括號、運算子優先權 |
| `pwd` | 列印工作目錄 | `std::env::current_dir()` |
| `tty` | 回報終端機名稱 | `ttyname_r()` syscall，非 tty 時 exit 1 |
| `logname` | 回報登入名稱 | `getlogin()` 搭配 `LOGNAME`/`USER` fallback |
| `logger` | 寫入系統日誌 | `libc::syslog(LOG_USER|LOG_INFO, ...)` + stderr |

## 實作細節

- `printf` 支援 POSIX 要求的 `%b`（跳脫處理）、`%d`/`%i`/`%u`/`%o`/`%x`/`%X`/`%f`/`%F`/`%e`/`%E`/`%g`/`%G`/`%s`/`%c`/`%%`、旗標（`-+ #0`）、寬度與精度（含 `*`）、八進位與十六進位跳脫序列
- `expr` 實作遞迴下降解析器，支援：`|` → `&` → `=` `!=` `>` `>=` `<` `<=` → `+` `-` → `*` `/` `%` → `(` `)`；僅整數（`i64`）
- `tty` 使用 `libc::ttyname_r` 查詢實際終端機路徑；若 `TERM` 未設定則回報 `not a tty`
- `logname` 先嘗試 `libc::getlogin()`，再 fallback 至環境變數
- `logger` 呼叫 `libc::syslog` 並同時輸出至 stderr（POSIX 行為）

## 整合測試

21 個新測試案例，涵蓋：
- `printf`：基本字串、`%s`、`%d`、`%x`、`%%`、`\\n`、`\\t` 跳脫
- `expr`：加減乘除、取模、等於、不等於、大於、邏輯或、邏輯與
- `pwd`：路徑格式驗證
- `tty`：非 tty 情境（exit 1）
- `logname`、`logger`：基本功能驗證
