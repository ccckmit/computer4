## 測試覆蓋

現有測試二進位：
- `/_fs` — 基礎檔案系統（open/close/read/write）
- `/_pipe` — Pipe 創建和通訊
- `/_proc` — Process 基本操作
- `/_fd` — 檔案描述符操作
- `/_sbrk` — 記憶體配置（brk/sbrk）
- `/_cow` — Copy-on-write fork
- `/_net` — 網路（UDP Socket）
- `/_posix_base` — POSIX 基礎（umask/exit/wait/fork）
- `/_posix_file` — 檔案操作（open/close/read/write/lseek/link/unlink/mkdir/...)
- `/_posix_mmap` — 記憶體映射（mmap/munmap/mprotect）
- `/_posix_proc` — 程序管理（getpid/getppid/getpgid/setsid/nice）
- `/_posix_time` — 時間 syscalls（clock_gettime/nanosleep）
- `/_posix_perm` — 權限模型（uid/gid/chmod/chown/umask）
- `/_posix_tty` — 終端機操作（tcgetattr/tcsetattr）
- `/_posix_vfs` — VFS 掛載（mount/umount/ProcFs/DevFs）
