# v0.4 — 檔案 I/O 擴充 (File I/O Expansion)

## 任務

- [x] **`sys_lseek`** — SEEK_SET / SEEK_CUR / SEEK_END，回傳新偏移量
- [x] **`sys_truncate`** — 路徑截斷檔案（重設大小為 0）
- [x] **`sys_ftruncate`** — fd 截斷檔案
- [x] **`sys_getdents`** — 目錄讀取，回傳 `Dirent`（inum + name）
- [x] **`sys_symlink`** / **`sys_readlink`** — 建立/讀取符號連結（以 File inode 儲存路徑）
- [x] **`sys_access`** — F_OK 檢查檔案存在
- [x] **`sys_fcntl`** — F_DUPFD / F_GETFD / F_SETFD / F_GETFL
- [x] **`sys_dup2`** — 複製 fd 到指定編號
- [x] **`Stat` 擴充** — 新增 `mode`、`uid`、`gid`、`blksize`、`blocks`、時間欄位
- [x] **`mode` 常數** — `mode::from_type()` 由 InodeType 推導 POSIX st_mode
- [x] **errno 擴充** — ESPIPE(29)、EOVERFLOW(75)、ENXIO(6)、EACCES(13) 等
- [x] **`_posix_file` 測試** — 7 項子測試：lseek、lseek_pipe、getdents、symlink/readlink、access、fcntl(F_DUPFD)、dup2

## 交付標準

- [x] `./test.sh` — 原有 9 項 + `_posix_file` 全部通過

- [x] `fcntl(F_DUPFD)` 可複製 fd
