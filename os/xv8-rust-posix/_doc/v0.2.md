## 回傳慣例（不變）

```
a0 = 0          → 成功
a0 = -(errno)   → 失敗（負值）
```

使用者空間解碼方式不變：`check(ret)` 若 `ret < 0` 則回傳 `Err(Errno::from((-ret) as u16))`。

## syscall_table 設計

```rust
type SyscallHandler = fn(&SyscallArgs) -> Result<usize, Errno>;

const SYSCALL_TABLE: [Option<SyscallHandler>; 64] = {
    let mut table: [Option<SyscallHandler>; 64] = [None; 64];
    table[1]  = Some(sys_fork);
    table[2]  = Some(sys_exit);
    table[3]  = Some(sys_wait);
    // ...
    table
};
```

`syscall()` 改為：
```rust
let handler = SYSCALL_TABLE[trapframe.a7]
    .ok_or(Errno::ENOSYS)?;
handler(&args)
```

## 破壞性變更

- `SysError` enum 完全移除，使用者程式中的 `SysError::Xxx` 須改為 `Errno::Xxx`
- `kernel::abi::SysError` 不再存在，改為 `kernel::abi::Errno`
