# v4.12 - Client SDK + Examples

## 目標

建立獨立的 Client SDK，讓應用程式可以遠端連線操作 db6，同時提供豐富的範例展示各種功能。

## Client SDK 設計

```rust
// src/client/mod.rs
pub struct Db6Client {
    connection: Connection,
}

impl Db6Client {
    pub async fn connect(addr: &str) -> Result<Self>;
    
    // Queue operations
    pub async fn enqueue(&self, queue: &str, data: Vec<u8>) -> Result<String>;
    pub async fn dequeue(&self, queue: &str) -> Result<Option<Message>>;
    pub async fn ack(&self, queue: &str, msg_id: &str) -> Result<()>;
    
    // PubSub operations
    pub async fn publish(&self, channel: &str, data: Vec<u8>) -> Result<String>;
    pub async fn subscribe(&self, channel: &str) -> Result<Receiver<Message>>;
    
    // SQL operations
    pub async fn execute(&self, sql: &str) -> Result<SqlResult>;
    pub async fn query(&self, sql: &str) -> Result<Vec<Row>>;
    
    // FTS operations
    pub async fn search(&self, index: &str, query: &str) -> Result<Vec<SearchResult>>;
}
```

## 實作重點

1. **Connection 封裝** - TCP/Unix socket 連線
2. **Request/Response 協議** - JSON 或二進制協定
3. **Reconnection 機制** - 自動重連
4. **Connection Pool** - 連線池（可選）
