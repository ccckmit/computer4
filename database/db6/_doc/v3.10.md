# v3.10 - GROUP BY / HAVING 支援

## 日期：2026-05-17

## 目標

將 query/ API 中的 GROUP BY / HAVING 功能整合到 SQL Executor，讓 SQL 語法也能支援聚合查詢。

## 背景

### query/ API 已有 GROUP BY 支援

```rust
// query/ API 支援 GROUP BY
db.select("*")
    .from("users")
    .group_by("city")
    .execute();
```

### SQL API 原本沒有 GROUP BY

```sql
-- 之前 SQL 無法使用 GROUP BY
SELECT city, COUNT(*) FROM users GROUP BY city;  -- 不支援
```

### v3.10 目標：SQL 支援 GROUP BY / HAVING

```sql
-- 現在 SQL 也支援 GROUP BY 了
SELECT age, COUNT(*) FROM users GROUP BY age;
SELECT city, COUNT(*) FROM users GROUP BY @.city HAVING COUNT(*) > 1;
```

## 語法設計

### GROUP BY 語法

```sql
-- 基本 GROUP BY
SELECT age, COUNT(*) FROM users GROUP BY age;

-- GROUP BY JSON 欄位
SELECT @.city, COUNT(*) FROM users GROUP BY @.city;

-- 多個 GROUP BY 欄位（可選）
SELECT age, city, COUNT(*) FROM users GROUP BY age, @.city;
```

### HAVING 語法

```sql
-- HAVING 過濾聚合結果
SELECT age, COUNT(*) FROM users GROUP BY age HAVING COUNT(*) > 1;

-- HAVING 與 WHERE 一起使用
SELECT age, COUNT(*) FROM users WHERE @.city = '台北' GROUP BY age HAVING COUNT(*) >= 2;

-- HAVING 支援的運算子
SELECT age, SUM(*) FROM users GROUP BY age HAVING SUM(*) >= 100;
SELECT age, AVG(*) FROM users GROUP BY age HAVING AVG(*) > 25.0;
SELECT age, MIN(*) FROM users GROUP BY age HAVING MIN(*) < 20;
SELECT age, MAX(*) FROM users GROUP BY age HAVING MAX(*) > 50;
```

## 支援的聚合函數

| 函數 | 說明 | 範例 |
|------|------|------|
| COUNT(*) | 計算數量 | SELECT COUNT(*) FROM users |
| SUM(*) | 總和 | SELECT SUM(age) FROM users |
| AVG(*) | 平均值 | SELECT AVG(age) FROM users |
| MIN(*) | 最小值 | SELECT MIN(age) FROM users |
| MAX(*) | 最大值 | SELECT MAX(age) FROM users |

## 實作方式

### 1. 新增 `apply_group_by` 函數

```rust
// src/sql/executor/executor.rs

fn apply_group_by(
    rows: Vec<(Vec<u8>, Vec<u8>)>,
    group_by: &[Expr],
    having: &Option<Expr>,
    columns: &[SelectItem],
) -> Result<Vec<(Vec<u8>, Vec<u8>)>> {
    use std::collections::HashMap;

    // 1. 按 group_by 表達式分組
    let mut groups: HashMap<String, Vec<(Vec<u8>, Vec<u8>)>> = HashMap::new();
    for (k, v) in rows {
        let group_key = extract_group_key(&group_by[0], &k, &v);
        groups.entry(group_key).or_insert_with(Vec::new).push((k, v));
    }

    // 2. 對每個群組計算聚合函數
    for (group_key, group_rows) in groups {
        let value = compute_aggregate(&group_rows, columns);
        // ... 處理 HAVING ...
        result.push((group_key.into_bytes(), value.into_bytes()));
    }

    Ok(result)
}
```

### 2. 修改 `execute_select` 呼叫 GROUP BY

```rust
fn execute_select(&mut self, select: &SelectStmt) -> Result<ResultSet> {
    // ... 掃描並過濾 ...

    // GROUP BY 處理
    if !select.group_by.is_empty() {
        rows = apply_group_by(rows, &select.group_by, &select.having, &select.columns)?;
    }

    // ... ORDER BY, LIMIT ...
}
```

### 3. HAVING 條件評估

```rust
fn eval_having(having: &Expr, agg_name: &str, agg_value: f64) -> bool {
    match having {
        Expr::BinOp { left, op, right } => {
            if let Expr::Function { name, .. } = left.as_ref() {
                if name.eq_ignore_ascii_case(agg_name) {
                    let rhs = extract_numeric(right);
                    return match op {
                        BinOp::Gt => agg_value > rhs,
                        BinOp::GtEq => agg_value >= rhs,
                        BinOp::Lt => agg_value < rhs,
                        BinOp::LtEq => agg_value <= rhs,
                        BinOp::Eq => (agg_value - rhs).abs() < 1e-9,
                        BinOp::NotEq => (agg_value - rhs).abs() >= 1e-9,
                        _ => true,
                    };
                }
            }
            true
        }
        _ => true,
    }
}
```

## API 對照

| query/ API | SQL API |
|------------|---------|
| `.group_by("city")` | `GROUP BY city` |
| `.group_by("@.city")` | `GROUP BY @.city` |
| (HAVING via filter) | `HAVING COUNT(*) > 1` |

## 測試案例

### 基本 GROUP BY

```rust
#[test]
fn test_group_by_basic() {
    let mut exec = Executor::new(Box::new(engine));
    exec.execute("INSERT INTO users VALUES ('{\"age\":30}')").unwrap();
    exec.execute("INSERT INTO users VALUES ('{\"age\":25}')").unwrap();
    exec.execute("INSERT INTO users VALUES ('{\"age\":30}')").unwrap();

    let result = exec.execute("SELECT age, COUNT(*) FROM users GROUP BY age").unwrap();
    assert_eq!(result.rows.len(), 2); // age=30 (2), age=25 (1)
}
```

### HAVING 條件

```rust
#[test]
fn test_group_by_having() {
    let mut exec = Executor::new(Box::new(engine));
    // ... insert data ...

    let result = exec.execute("SELECT age, COUNT(*) FROM users GROUP BY age HAVING COUNT(*) > 1").unwrap();
    // 只返回 count > 1 的群組
}
```

### JSON Path GROUP BY

```rust
#[test]
fn test_group_by_json_path() {
    let mut exec = Executor::new(Box::new(engine));
    // ... insert data with city field ...

    let result = exec.execute("SELECT @.city, COUNT(*) FROM users GROUP BY @.city").unwrap();
    // 按 city 分組
}
```

## 不做的事項

- ❌ DISTINCT 聚合（如 `SELECT COUNT(DISTINCT city)`）
- ❌ GROUP_CONCAT 聚合函數
- ❌ 百分比聚合（如 `SELECT PERCENTILE_CONT(0.5)`）
- ❌ CUBE / ROLLUP

## 附錄：完整範例

```sql
-- 建立測試資料
CREATE TABLE users;
INSERT INTO users VALUES ('{"name":"Alice","age":30,"city":"Taipei"}');
INSERT INTO users VALUES ('{"name":"Bob","age":25,"city":"Kaohsiung"}');
INSERT INTO users VALUES ('{"name":"Charlie","age":30,"city":"Taipei"}');
INSERT INTO users VALUES ('{"name":"Diana","age":28,"city":"Taichung"}');

-- 基本聚合
SELECT age, COUNT(*) FROM users GROUP BY age;
-- 結果: 30 -> 2, 25 -> 1, 28 -> 1

-- HAVING 過濾
SELECT age, COUNT(*) FROM users GROUP BY age HAVING COUNT(*) > 1;
-- 結果: 30 -> 2

-- JSON Path GROUP BY
SELECT @.city, COUNT(*) FROM users GROUP BY @.city;
-- 結果: Taipei -> 2, Kaohsiung -> 1, Taichung -> 1

-- HAVING 與 JSON Path
SELECT @.city, COUNT(*) FROM users GROUP BY @.city HAVING COUNT(*) >= 2;
-- 結果: Taipei -> 2
```
