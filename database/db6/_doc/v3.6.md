## 新功能

### MapReduce 查詢介面

在 Query Module 中新增 MapReduce 功能，支援資料轉換與聚合：

```rust
// 設計：table("users").map(...).reduce(...).execute()

// Map: 轉換每個 key-value 對
let result = db.table("users")
    .map(|k, v| (k.to_vec(), String::from_utf8_lossy(v).to_uppercase().into_bytes()))
    .unwrap()
    .reduce(|acc, k, v| { /* 聚合 */ })
    .execute()?;

// Count 聚合
let result = db.table("users")
    .map(|k, v| (k.to_vec(), v.to_vec()))
    .unwrap()
    .reduce(|acc, _, _| {
        let count = if acc.is_empty() { 0 } else { ... };
        (count + 1).to_string().into_bytes()
    })
    .execute()?;

// Sum 聚合
let result = db.table("numbers")
    .map(|k, v| (k.to_vec(), v.to_vec()))
    .unwrap()
    .reduce(|acc, _, v| {
        let sum: i32 = if acc.is_empty() { 0 } else { ... };
        let val: i32 = String::from_utf8_lossy(v).parse().unwrap_or(0);
        (sum + val).to_string().into_bytes()
    })
    .execute()?;
```

## API 設計

```rust
// TableQuery 新增 map() 方法
impl<'a> TableQuery<'a> {
    pub fn map<F>(self, f: F) -> Result<MapReduceQuery<'a>>
    where
        F: Fn(&[u8], &[u8]) -> (Vec<u8>, Vec<u8>) + Send + Sync + 'static;
}

// MapReduceQuery 提供 reduce() 和 execute()
pub struct MapReduceQuery<'a> {
    db: &'a mut Db,
    table_name: String,
    table_id: u32,
    data: Vec<(Vec<u8>, Vec<u8>)>,
    reducer: Option<Box<dyn FnMut(Vec<u8>, &[u8], &[u8]) -> Vec<u8> + Send + Sync>>,
}

impl MapReduceQuery {
    pub fn reduce<F>(self, f: F) -> Self
    where
        F: FnMut(Vec<u8>, &[u8], &[u8]) -> Vec<u8> + Send + Sync + 'static;

    pub fn execute(self) -> Result<Vec<(Vec<u8>, Vec<u8>)>>;
    pub fn execute_scalar(self) -> Result<String>;
}
```

## 設計考量

- 使用 `table("users").map(...).reduce(...).execute()` 風格
- 與現有 fluent chain 風格一致（如 `table("users").put().put().flush()`）
- `map()` 執行映射並返回 `MapReduceQuery`，`reduce()` 設定 reducer
- `execute()` 執行 reduce 並返回結果
