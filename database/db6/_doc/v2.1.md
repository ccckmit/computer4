## CLI 使用方式

```bash
# 編譯
cargo build --release

# 執行 REPL
./target/release/db6

# 或用 cargo 執行
cargo run --release
```

REPL 命令：
- `.quit` / `.exit` - 離開
- `.help` - 顯示說明
- `.engine` - 顯示目前的引擎
