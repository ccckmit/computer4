# v3.1 完成項目（Query Fluent Interface 完整版）

## 日期：2026-05-16

## 新功能

### 1. INSERT Fluent Interface

```rust
let count = db.insert()
    .into_table("users")
    .values(vec![("1", "Alice"), ("2", "Bob")])
    .execute()?;
```

### 2. DELETE Fluent Interface

```rust
let count = db.delete()
    .from("users")
    .where_("value = Bob")
    .execute()?;
```

### 3. UPDATE Fluent Interface

```rust
let count = db.update("users")
    .set_value("Robert")
    .where_("value = Bob")
    .execute()?;
```

### 4. WHERE Filtering（增強）

在 `SelectQuery` 中實作 WHERE 子句解析與資料過濾：

```rust
let result = db.select("key, value")
    .from("users")
    .where_("value = Bob")
    .execute()?;
```

支援的運算子：
- `=`, `==` - 等於
- `!=` - 不等於
- `>` - 大於
- `>=` - 大於等於
- `<` - 小於
- `<=` - 小於等於
- `LIKE` - 模糊比對（支援 `%pattern%`, `pattern%`, `%pattern`）

WHERE 可過濾的欄位：
- `key` - 鍵欄位
- `value` - 值欄位（預設）
