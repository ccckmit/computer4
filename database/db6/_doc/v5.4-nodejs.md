# v5.4 Node.js Client (db6nodejs)

## 完成項目

- npm 套件 `db6nodejs`
- REST API 客戶端
- 同步 async/await 介面
- 完整 KV 操作支援

## 安裝

```bash
npm install db6nodejs
```

## 使用方式

```javascript
import { Client } from 'db6nodejs';

const db = new Client('http://localhost:50052');

// Health check
await db.health();

// KV operations
await db.put(1, 'key', 'value');
const [value, found] = await db.get(1, 'key');
await db.delete(1, 'key');

// Batch operations
await db.batchPut(1, [['k1', 'v1'], ['k2', 'v2']]);
const results = await db.scan(1, 'k', 'k\xff');

// Stats
const stats = await db.stats();

// Range delete
await db.rangeDelete(1, 'start', 'end');

await db.close();
```

## API

### new Client(baseUrl, options?)

| Method | Description |
|--------|-------------|
| health() | Check server health, returns Promise<boolean> |
| put(tableId, key, value) | Put a key-value pair |
| get(tableId, key) | Get value, returns Promise<[value, found]> |
| delete(tableId, key) | Delete a key |
| batchPut(tableId, items) | Batch put [[key, value], ...] |
| scan(tableId, start, end) | Scan range, returns Promise<[[key, value], ...]> |
| stats() | Get statistics, returns Promise<object> |
| rangeDelete(tableId, start, end) | Delete range |
| close() | Close connection |
