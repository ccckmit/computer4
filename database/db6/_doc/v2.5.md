# v2.5 - Memory Engine 模組化

## 完成日期
2026-05-16

## 完成項目

### 1. 模組化重構

將 `memory.rs` 重構為 `memory/` 目錄：
- `memory/mod.rs` - 模組入口
- `memory/hash.rs` - HashMemoryEngine (Redis-like)
- `memory/btree.rs` - BTreeMemoryEngine (SQLite-like)

### 2. 兩種引擎

#### HashMemoryEngine
- 使用 HashMap，O(1) 操作
- 不支援 ORDER BY / 範圍掃描
- 適合：快速 KV 操作、快取

#### BTreeMemoryEngine
- 使用 BTreeMap，O(log n) 操作
- 支援 ORDER BY / 範圍掃描
- 適合：SQL 查詢、測試

### 3. CLI 更新

```bash
.engine memory-hash  # HashMap 版本
.engine memory-btree # BTreeMap 版本
```
