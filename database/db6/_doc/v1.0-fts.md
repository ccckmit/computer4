# v1.0 - FTS 全文檢索

## 完成日期
2026-05-16

## 完成項目

### FTS 全文檢索
- [x] FtsIndex - 基於 KV 介面的倒排索引
- [x] CjkTokenizer - CJK 雙元分詞
- [x] EnglishTokenizer - 英文小寫分詞
- [x] search() 查詢功能
- [x] 支援所有儲存引擎 (Memory/BTree/LSM)

### Tokenizer 特性
- CJK: "資料庫系統" → ["資料", "料庫", "庫系", "系統"]
- English: "Hello World" → ["hello", "world"]

## 架構

```
FTS (基於 KV)
  │
  ├── FtsIndex<E: StorageEngine>
  │     ├── insert(doc_id, text)
  │     └── search(query) -> Vec<doc_id>
  │
  └── Tokenizers
        ├── CjkTokenizer (雙元分詞)
        └── EnglishTokenizer

KV Interface (三引擎通用)
  ├── MemoryEngine
  ├── BTreeEngine
  └── LsmEngine
```
