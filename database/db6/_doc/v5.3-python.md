## 安裝

```bash
pip install db6py
```

With WebSocket:
```bash
pip install db6py[websocket]
```

## 使用方式

```python
from db6py import Client

# REST client
db = Client("http://localhost:50052")

# Health check
db.health()

# KV operations
db.put(1, "key", "value")
value, found = db.get(1, "key")
db.delete(1, "key")

# Batch operations
db.batch_put(1, [("k1", "v1"), ("k2", "v2")])
results = db.scan(1, "k", "k\xff")

# Stats
stats = db.stats()

# Range delete
db.range_delete(1, "start", "end")
```

## API

### Client(base_url)

| Method | Description |
|--------|-------------|
| health() | Check server health, returns bool |
| put(table_id, key, value) | Put a key-value pair |
| get(table_id, key) | Get value, returns (value, found) |
| delete(table_id, key) | Delete a key |
| batch_put(table_id, items) | Batch put [(key, value), ...] |
| scan(table_id, start, end) | Scan range, returns [(key, value), ...] |
| stats() | Get statistics |
| range_delete(table_id, start, end) | Delete range |
