# v3.7 - JSON 過濾支援

## 日期：2026-05-16

## 目標

在 WHERE 子句中支援簡潔的 JSON 路徑語法，SELECT 結果回傳完整 JSON。

## API 設計

### 語法

```rust
// filter() 或 where_() 皆可使用
db.select("*")
    .from("users")
    .filter("$.age > 25")
    .execute();

// where_() 是 filter() 的別名，兩者行為相同
db.select("*")
    .from("users")
    .where_("$.age > 25")
    .execute();
```

### 支援運算子

| 運算子 | 說明 | 範例 |
|--------|------|------|
| `>` | 大於 | `$.age > 25` |
| `<` | 小於 | `$.age < 30` |
| `>=` | 大於等於 | `$.age >= 25` |
| `<=` | 小於等於 | `$.age <= 30` |
| `=` | 等於 | `$.name = 'Alice'` |
| `!=` | 不等於 | `$.name != 'Bob'` |
| `LIKE` | 模糊匹配 | `$.name LIKE 'Ali%'` |

### 支援 JSON 路徑

| 語法 | 說明 | 範例 |
|------|------|------|
| `$.field` | 物件欄位 | `$.name` |
| `$.nested.field` | 巢狀欄位 | `$.address.city` |
| `$[0]` | 陣列索引 | `$[0]` |
| `$.items[0]` | 陣列物件 | `$.items[0].name` |

### 查詢結果

回傳完整 JSON，不是提取的單一值：

```rust
// 資料
db.table("users").put(b"1", r#"{"name":"Alice","age":30}"#.as_bytes()).unwrap();
db.table("users").put(b"2", r#"{"name":"Bob","age":25}"#.as_bytes()).unwrap();
db.table("users").put(b"3", r#"{"name":"Charlie","age":35}"#.as_bytes()).unwrap();

// 查詢 age > 25
let result = db.select("*")
    .from("users")
    .filter("$.age > 25")
    .execute()
    .unwrap();

// result.rows:
// [["1", "{\"name\":\"Alice\",\"age\":30}"],
//  ["3", "{\"name\":\"Charlie\",\"age\":35}"]]
```

## 實作方式

### 1. 偵測 JSON 路徑語法

```rust
// 當 WHERE 條件以 $ 開頭時，視為 JSON 路徑過濾
if condition.starts_with("$.") || condition.starts_with("$[") {
    return filter_json_path(rows, condition);
}
```

### 2. filter() 和 where_() 兩者皆可使用

```rust
pub fn filter(&mut self, condition: &str) -> &mut Self {
    self.where_clause = Some(condition.to_string());
    self
}

pub fn where_(&mut self, condition: &str) -> &mut Self {
    self.filter(condition)  // 委託給 filter
}
```

## Functional Programming 三元素

現在 Query Module 支援 `map` / `reduce` / `filter`：

```rust
// Map - 轉換
db.table("users")
    .map(|k, v| (k.to_vec(), transform(v)))
    .unwrap()
    .execute()

// Filter - 過濾
db.select("*")
    .from("users")
    .filter("$.age > 25")
    .execute()

// Reduce - 聚合
db.table("users")
    .map(|k, v| (k.to_vec(), v.to_vec()))
    .unwrap()
    .reduce(|acc, _, v| accumulate(acc, v))
    .execute()
```
