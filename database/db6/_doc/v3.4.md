# v3.4 完成項目（GROUP BY 支援）

## 日期：2026-05-16

## 新功能

### GROUP BY 支援

```rust
let result = db.select("COUNT(*), value")
    .from("users")
    .group_by("value")
    .execute()?;
```

### Engine Capability 檢查

使用 `CanGroupBy` capability trait，只有 BTree 類型引擎支援 GROUP BY：
- `memory-btree` - 支援
- `btree` - 支援  
- `memory-hash` - **不支援**，編譯/執行時會報錯
- `lsm` - **不支援**

### 支援的 Aggregate Functions

- `COUNT(*)` - 計算群組內的項目數
- `SUM(column)` - 總和
- `AVG(column)` - 平均
- `MIN(column)` - 最小值
- `MAX(column)` - 最大值
