# v0.3 - LSM Engine

## 完成日期
2026-05-16

## 完成項目

### LSM Engine
- [x] MemTable (in-memory write buffer with BTreeMap)
- [x] Tombstone support for deletions
- [x] Basic KV operations (get/put/delete/scan)
- [x] Limited transaction support (single table only)
- [x] Thread-safe with RwLock

### 新增模組
- [x] `src/engine/lsm.rs` - Complete LsmEngine implementation

## 架構

```
User API
  |
  +-- KvStore (unified KV interface)
  |     +-- MemoryEngine (BTreeMap, in-memory)
  |     +-- BTreeEngine (BTree, in-memory, transaction support)
  |     +-- LsmEngine (MemTable, single-table transactions)
```

## 測試結果
```
16 tests passed (5 memory + 5 btree + 6 lsm)
```

## LsmEngine 特性
- 只支援 table_id = 1（單一表格操作）
- 不支援多表格（不同於 Memory/BTree）
- 有限交易支援（single table only）
- Tombstone 機制處理刪除

## v0.x 版本總結

| 版本 | 引擎 | 特性 |
|------|------|------|
| v0.1 | Memory | BTreeMap, in-memory, 無交易 |
| v0.2 | BTree | BTree, 完整交易, 執行緒安全 |
| v0.3 | LSM | MemTable, single-table transaction |
