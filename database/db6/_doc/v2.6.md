### 1. Capability Module

建立 `src/engine/capability.rs`：
```rust
pub trait CanOrderBy: StorageEngine {}  // ORDER BY 支援
pub trait CanJoin: StorageEngine {}     // JOIN 支援
pub trait CanFts: StorageEngine {}      // FTS 支援
pub trait CanTransaction: StorageEngine {} // 交易支援
pub trait CanScan: StorageEngine {}     // 範圍掃描支援
pub trait CanBatch: StorageEngine {}   // 批量操作支援
```

### 3. SqlExecutor

新增泛型 SQL 執行器，支援編譯時期檢查：
```rust
pub struct SqlExecutor<E: StorageEngine> {
    engine: E,
}

impl<E: StorageEngine + CanOrderBy> SqlExecutor<E> {
    pub fn execute_order_by(&mut self, sql: &str) -> Result<ResultSet> {
        // 只有支援 ORDER BY 的引擎才能編譯
    }
}
```

### 4. 編譯時期錯誤範例

```rust
// 這會在編譯時期失敗
use db6::{SqlExecutor, engine::HashMemoryEngine};

let mut exec: SqlExecutor<HashMemoryEngine> = SqlExecutor::new(engine);
exec.execute("SELECT * FROM t ORDER BY id");

// 編譯錯誤:
// the trait `CanOrderBy` is not implemented for `HashMemoryEngine`
```
