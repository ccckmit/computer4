# v4.0 - Message Queue (msgq) 模組

## 日期：2026-05-17

## 目標

基於現有的 `kv/` 模組，建立一個 Message Queue 訊息佇列系統。提供可靠、可持久化的佇列服務，支援 FIFO 順序、訊息確認機制、 Visibility Timeout 等功能。

## 架構設計

### 模組位置

```
src/
├── kv/                    # 現有 KV 儲存（基礎層）
│   └── mod.rs
├── msgq/                  # 新增 Message Queue 模組
│   ├── mod.rs             # 統一出口
│   ├── queue.rs           # Queue 核心實作
│   ├── message.rs         # Message 結構
│   ├── error.rs           # 錯誤類型
│   └── tests.rs           # 測試
```

### 核心概念

1. **Queue（佇列）** — 訊息的邏輯容器，每個 Queue 有名稱
2. **Message（訊息）** — 實際傳遞的資料內容
3. **Enqueue（入隊）** — 將訊息放入佇列尾部
4. **Dequeue（出隊）** — 從佇列头部取出訊息
5. **Ack（確認）** — 訊息處理完成，刪除訊息
6. **Nack（未確認）** — 訊息處理失敗，重新放回佇列
7. **Visibility Timeout** — 訊息可見性逾時（避免重複處理）

## Key 設計

使用 KV 儲存訊息，Key 格式：

```
queue:{queue_name}:meta        -> Queue 元資料（JSON）
queue:{queue_name}:msg:{id}   -> 訊息內容（JSON）
queue:{queue_name}:index      -> 訊息 ID 列表（sorted by enqueue time）
queue:{queue_name}:inflight    -> 處理中的訊息 ID（visibility timeout 管理）
```

### Message ID 格式

```
timestamp:uuid
例如: 1716000000000:a1b2c3d4e5f6
```

### 2. Message 結構（message.rs）

```rust
use serde::{Deserialize, Serialize};

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Message {
    pub id: String,
    pub payload: Vec<u8>,
    pub enqueued_at: u64,      // Unix timestamp (milliseconds)
    pub delivery_count: u32,    // 遞送次數
    pub visibility_timeout: u64, // 可見性逾時（秒）
    pub visible_after: u64,    //  visible after this timestamp
    pub metadata: Option<String>, // 可選的中繼資料（JSON）
}

impl Message {
    pub fn new(payload: Vec<u8>, visibility_timeout: u64) -> Self {
        let now = std::time::SystemTime::now()
            .duration_since(std::time::UNIX_EPOCH)
            .unwrap()
            .as_millis() as u64;

        let id = format!("{}:{:x}", now, rand_uuid());

        Self {
            id,
            payload,
            enqueued_at: now,
            delivery_count: 0,
            visibility_timeout,
            visible_after: 0,
            metadata: None,
        }
    }

    pub fn is_visible(&self) -> bool {
        let now = std::time::SystemTime::now()
            .duration_since(std::time::UNIX_EPOCH)
            .unwrap()
            .as_millis() as u64;
        now >= self.visible_after
    }
}
```

### 3. Queue 核心實作（queue.rs）

```rust
use crate::kv::{KvEngine, KvStore};
use super::{message::Message, error::*, Result};
use std::sync::{Arc, RwLock};

pub struct Queue {
    name: String,
    engine: Arc<RwLock<KvEngine>>,
}

impl Queue {
    pub fn new(name: &str, engine: Arc<RwLock<KvEngine>>) -> Self {
        Self {
            name: name.to_string(),
            engine,
        }
    }

    /// Enqueue - 將訊息放入佇列
    pub fn enqueue(&mut self, payload: Vec<u8>, visibility_timeout: u64) -> Result<String> {
        let mut msg = Message::new(payload, visibility_timeout);
        let msg_id = msg.id.clone();

        // 序列化訊息
        let msg_json = serde_json::to_vec(&msg)
            .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;

        // 寫入訊息 Key
        let msg_key = format!("queue:{}:msg:{}", self.name, msg_id);
        self.engine.write().unwrap()
            .put(1, msg_key.as_bytes(), &msg_json)?;

        // 更新 index（以 timestamp 排序）
        let index_key = format!("queue:{}:index", self.name);
        let mut index = self.read_index()?;
        index.push(msg_id.clone());
        let index_json = serde_json::to_vec(&index)
            .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;
        self.engine.write().unwrap()
            .put(1, index_key.as_bytes(), &index_json)?;

        // 更新 meta
        self.update_meta(|m| m.total_enqueued += 1)?;

        Ok(msg_id)
    }

    /// Dequeue - 取出訊息（不可見狀態）
    pub fn dequeue(&mut self, wait_timeout_secs: u64) -> Result<Option<Message>> {
        // 讀取 index
        let index = self.read_index()?;
        let now = std::time::SystemTime::now()
            .duration_since(std::time::UNIX_EPOCH)
            .unwrap()
            .as_millis() as u64;

        // 遍歷找可見訊息
        for msg_id in &index {
            let msg_key = format!("queue:{}:msg:{}", self.name, msg_id);
            if let Some(data) = self.engine.read().unwrap()
                .get(1, msg_key.as_bytes())? {

                let mut msg: Message = serde_json::from_slice(&data)
                    .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;

                // 檢查可見性
                if msg.is_visible() {
                    // 檢查是否已在 inflight（避免重複領取）
                    if self.is_inflight(&msg_id)? {
                        continue;
                    }

                    // 標記為 inflight
                    self.add_inflight(&msg_id)?;

                    // 更新 delivery_count
                    msg.delivery_count += 1;
                    msg.visible_after = now + msg.visibility_timeout * 1000;

                    let updated_json = serde_json::to_vec(&msg)
                        .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;
                    self.engine.write().unwrap()
                        .put(1, msg_key.as_bytes(), &updated_json)?;

                    return Ok(Some(msg));
                }
            }
        }

        // 無可用訊息，輪詢等待
        if wait_timeout_secs > 0 {
            std::thread::sleep(std::time::Duration::from_secs(1));
            return self.dequeue(wait_timeout_secs - 1);
        }

        Ok(None)
    }

    /// Ack - 確認訊息處理完成
    pub fn ack(&mut self, msg_id: &str) -> Result<()> {
        let msg_key = format!("queue:{}:msg:{}", self.name, msg_id);

        // 檢查訊息是否存在
        if self.engine.read().unwrap().get(1, msg_key.as_bytes())?.is_none() {
            return Err(MsgqError::MessageNotFound(msg_id.to_string()));
        }

        // 從 index 移除
        self.remove_from_index(msg_id)?;

        // 從 inflight 移除
        self.remove_inflight(msg_id)?;

        // 刪除訊息
        self.engine.write().unwrap()
            .delete(1, msg_key.as_bytes())?;

        // 更新 meta
        self.update_meta(|m| m.completed += 1)?;

        Ok(())
    }

    /// Nack - 訊息處理失敗，重新放回佇列
    pub fn nack(&mut self, msg_id: &str) -> Result<()> {
        let msg_key = format!("queue:{}:msg:{}", self.name, msg_id);

        // 取得訊息
        let data = self.engine.read().unwrap()
            .get(1, msg_key.as_bytes())?
            .ok_or_else(|| MsgqError::MessageNotFound(msg_id.to_string()))?;

        let mut msg: Message = serde_json::from_slice(&data)
            .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;

        // 從 inflight 移除
        self.remove_inflight(msg_id)?;

        // 重設 visible_after 為現在
        let now = std::time::SystemTime::now()
            .duration_since(std::time::UNIX_EPOCH)
            .unwrap()
            .as_millis() as u64;
        msg.visible_after = now;

        // 寫回訊息
        let updated_json = serde_json::to_vec(&msg)
            .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;
        self.engine.write().unwrap()
            .put(1, msg_key.as_bytes(), &updated_json)?;

        // 更新 meta
        self.update_meta(|m| m.nacked += 1)?;

        Ok(())
    }

    /// Peek - 查看訊息（不移除）
    pub fn peek(&self) -> Result<Option<Message>> {
        let index = self.read_index()?;

        for msg_id in &index {
            let msg_key = format!("queue:{}:msg:{}", self.name, msg_id);
            if let Some(data) = self.engine.read().unwrap()
                .get(1, msg_key.as_bytes())? {
                let msg: Message = serde_json::from_slice(&data)
                    .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;
                if msg.is_visible() {
                    return Ok(Some(msg));
                }
            }
        }

        Ok(None)
    }

    /// Length - 取得佇列長度
    pub fn length(&self) -> Result<usize> {
        let index = self.read_index()?;
        Ok(index.len())
    }

    /// Purge - 清空佇列
    pub fn purge(&mut self) -> Result<()> {
        let index = self.read_index()?;

        // 刪除所有訊息
        for msg_id in &index {
            let msg_key = format!("queue:{}:msg:{}", self.name, msg_id);
            self.engine.write().unwrap()
                .delete(1, msg_key.as_bytes())?;
        }

        // 清空 index
        let index_key = format!("queue:{}:index", self.name);
        self.engine.write().unwrap()
            .delete(1, index_key.as_bytes())?;

        // 清空 inflight
        let inflight_key = format!("queue:{}:inflight", self.name);
        self.engine.write().unwrap()
            .delete(1, inflight_key.as_bytes())?;

        // 重置 meta
        self.reset_meta()?;

        Ok(())
    }

    // --- 內部輔助方法 ---

    fn read_index(&self) -> Result<Vec<String>> {
        let index_key = format!("queue:{}:index", self.name);
        if let Some(data) = self.engine.read().unwrap()
            .get(1, index_key.as_bytes())? {
            Ok(serde_json::from_slice(&data)
                .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?)
        } else {
            Ok(vec![])
        }
    }

    fn remove_from_index(&mut self, msg_id: &str) -> Result<()> {
        let index_key = format!("queue:{}:index", self.name);
        let mut index = self.read_index()?;
        index.retain(|id| id != msg_id);

        let index_json = serde_json::to_vec(&index)
            .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;
        self.engine.write().unwrap()
            .put(1, index_key.as_bytes(), &index_json)?;
        Ok(())
    }

    fn is_inflight(&self, msg_id: &str) -> Result<bool> {
        let inflight_key = format!("queue:{}:inflight", self.name);
        if let Some(data) = self.engine.read().unwrap()
            .get(1, inflight_key.as_bytes())? {
            let inflight: Vec<String> = serde_json::from_slice(&data)
                .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;
            Ok(inflight.contains(&msg_id.to_string()))
        } else {
            Ok(false)
        }
    }

    fn add_inflight(&mut self, msg_id: &str) -> Result<()> {
        let inflight_key = format!("queue:{}:inflight", self.name);
        let mut inflight = if let Some(data) = self.engine.read().unwrap()
            .get(1, inflight_key.as_bytes())? {
            serde_json::from_slice(&data)
                .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?
        } else {
            vec![]
        };
        inflight.push(msg_id.to_string());

        let inflight_json = serde_json::to_vec(&inflight)
            .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;
        self.engine.write().unwrap()
            .put(1, inflight_key.as_bytes(), &inflight_json)?;
        Ok(())
    }

    fn remove_inflight(&mut self, msg_id: &str) -> Result<()> {
        let inflight_key = format!("queue:{}:inflight", self.name);
        let mut inflight = if let Some(data) = self.engine.read().unwrap()
            .get(1, inflight_key.as_bytes())? {
            serde_json::from_slice(&data)
                .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?
        } else {
            vec![]
        };
        inflight.retain(|id| id != msg_id);

        let inflight_json = serde_json::to_vec(&inflight)
            .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;
        self.engine.write().unwrap()
            .put(1, inflight_key.as_bytes(), &inflight_json)?;
        Ok(())
    }

    #[derive(serde::Serialize, serde::Deserialize)]
    struct QueueMeta {
        total_enqueued: u64,
        completed: u64,
        nacked: u64,
    }

    fn update_meta<F: FnOnce(&mut QueueMeta)>(&mut self, f: F) -> Result<()> {
        let meta_key = format!("queue:{}:meta", self.name);
        let mut meta = if let Some(data) = self.engine.read().unwrap()
            .get(1, meta_key.as_bytes())? {
            serde_json::from_slice(&data)
                .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?
        } else {
            QueueMeta { total_enqueued: 0, completed: 0, nacked: 0 }
        };
        f(&mut meta);

        let meta_json = serde_json::to_vec(&meta)
            .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;
        self.engine.write().unwrap()
            .put(1, meta_key.as_bytes(), &meta_json)?;
        Ok(())
    }

    fn reset_meta(&mut self) -> Result<()> {
        let meta_key = format!("queue:{}:meta", self.name);
        let meta = QueueMeta { total_enqueued: 0, completed: 0, nacked: 0 };
        let meta_json = serde_json::to_vec(&meta)
            .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;
        self.engine.write().unwrap()
            .put(1, meta_key.as_bytes(), &meta_json)?;
        Ok(())
    }
}
```

### 4. Msgq 管理器（mod.rs）

```rust
//! Message Queue - 基於 KV 的訊息佇列系統

mod error;
mod message;
mod queue;

pub use error::{MsgqError, Result};
pub use message::Message;
pub use queue::Queue;

use crate::kv::{KvEngine, KvStore};
use std::sync::{Arc, RwLock};

/// Msgq 實例
pub struct Msgq {
    engine: Arc<RwLock<KvEngine>>,
}

impl Msgq {
    pub fn new(engine_type: &str) -> Result<Self> {
        let engine = KvEngine::new(engine_type)?;
        Ok(Self {
            engine: Arc::new(RwLock::new(engine)),
        })
    }

    pub fn open(engine_type: &str, path: &std::path::Path) -> Result<Self> {
        let engine = KvEngine::open(engine_type, path)?;
        Ok(Self {
            engine: Arc::new(RwLock::new(engine)),
        })
    }

    /// 取得或建立 Queue
    pub fn queue(&self, name: &str) -> Queue {
        Queue::new(name, self.engine.clone())
    }

    /// 列出所有 Queue
    pub fn list_queues(&self) -> Result<Vec<String>> {
        let start = b"queue:";
        let end = b"queue;";

        let results = self.engine.read().unwrap().scan(1, start, end)?;
        let mut queues = std::collections::HashSet::new();

        for (key, _) in results {
            let key_str = String::from_utf8_lossy(&key);
            if let Some(name) = key_str.strip_prefix("queue:") {
                if let Some(queue_name) = name.split(':').next() {
                    if !queue_name.is_empty() {
                        queues.insert(queue_name.to_string());
                    }
                }
            }
        }

        Ok(queues.into_iter().collect())
    }

    /// 刪除 Queue（含所有訊息）
    pub fn delete_queue(&mut self, name: &str) -> Result<()> {
        let queue = self.queue(name);
        queue.purge()?;
        Ok(())
    }

    /// 取得 Queue 統計資訊
    pub fn stats(&self, name: &str) -> Result<QueueStats> {
        let queue = self.queue(name);
        let meta_key = format!("queue:{}:meta", name);

        if let Some(data) = self.engine.read().unwrap().get(1, meta_key.as_bytes())? {
            #[derive(serde::Deserialize)]
            struct QueueMeta {
                total_enqueued: u64,
                completed: u64,
                nacked: u64,
            }
            let meta: QueueMeta = serde_json::from_slice(&data)
                .map_err(|e| MsgqError::InvalidFormat(e.to_string()))?;

            Ok(QueueStats {
                name: name.to_string(),
                length: queue.length()?,
                total_enqueued: meta.total_enqueued,
                completed: meta.completed,
                nacked: meta.nacked,
            })
        } else {
            Ok(QueueStats {
                name: name.to_string(),
                length: 0,
                total_enqueued: 0,
                completed: 0,
                nacked: 0,
            })
        }
    }
}

use serde::Serialize;

#[derive(Debug, Serialize)]
pub struct QueueStats {
    pub name: String,
    pub length: usize,
    pub total_enqueued: u64,
    pub completed: u64,
    pub nacked: u64,
}
```

## 使用範例

```rust
use db6::msgq::Msgq;

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // 建立 Msgq 實例
    let mut msgq = Msgq::new("memory")?;

    // 建立佇列
    let mut queue = msgq.queue("orders");

    // Enqueue - 放入訊息
    let msg_id = queue.enqueue(b"Order #12345 data", 30)?;
    println!("Enqueued message: {}", msg_id);

    // Dequeue - 取出訊息
    if let Some(msg) = queue.dequeue(0)? {
        println!("Dequeued: {} - {:?}", msg.id, String::from_utf8_lossy(&msg.payload));

        // 處理完成，確認訊息
        queue.ack(&msg.id)?;
    }

    // 查看佇列長度
    println!("Queue length: {}", queue.length());

    // 統計資訊
    let stats = msgq.stats("orders")?;
    println!("Stats: {:?}", stats);

    Ok(())
}
```

## 測試案例

```rust
#[test]
fn test_msgq_basic() {
    let msgq = Msgq::new("memory").unwrap();
    let mut queue = msgq.queue("test");

    // Enqueue
    let id1 = queue.enqueue(b"msg1", 30).unwrap();
    let id2 = queue.enqueue(b"msg2", 30).unwrap();

    assert_eq!(queue.length().unwrap(), 2);

    // Dequeue
    let msg = queue.dequeue(0).unwrap().unwrap();
    assert!(msg.payload == b"msg1" || msg.payload == b"msg2");

    // Ack
    queue.ack(&msg.id).unwrap();
    assert_eq!(queue.length().unwrap(), 1);
}

#[test]
fn test_msgq_visibility_timeout() {
    let msgq = Msgq::new("memory").unwrap();
    let mut queue = msgq.queue("test");

    // Enqueue with 1 second visibility timeout
    queue.enqueue(b"msg", 1).unwrap();

    // Dequeue - 取得訊息
    let msg = queue.dequeue(0).unwrap().unwrap();

    // 立即再次 dequeue，不應取得同一訊息（inflight）
    let msg2 = queue.dequeue(0).unwrap();
    assert!(msg2.is_none() || msg2.unwrap().id != msg.id);

    // 等待超過 visibility timeout
    std::thread::sleep(std::time::Duration::from_secs(2));

    // 現在應該可以再次取得
    let msg3 = queue.dequeue(0).unwrap().unwrap();
    assert_eq!(msg3.id, msg.id);
}

#[test]
fn test_msgq_nack() {
    let msgq = Msgq::new("memory").unwrap();
    let mut queue = msgq.queue("test");

    let id = queue.enqueue(b"msg", 30).unwrap();

    // Dequeue
    let msg = queue.dequeue(0).unwrap().unwrap();
    assert_eq!(msg.delivery_count, 1);

    // Nack - 重新放回佇列
    queue.nack(&msg.id).unwrap();

    // 再次取得，delivery_count 應該是 2
    let msg2 = queue.dequeue(0).unwrap().unwrap();
    assert_eq!(msg2.delivery_count, 2);
}
```

## 現有挑戰與未來優化

1. **並發處理**
   - 目前使用 RwLock 保護 engine，多執行緒並發 enqueue/dequeue 可能需要更精細的鎖控制
   - 未來可考慮使用事務（BTree/LSM）確保 atomicity

2. **Visibility Timeout 實作**
   - 目前用簡單的輪詢，未來可加入 background worker 定期清理過期的 inflight 狀態

3. **多 Consumer 支援**
   - 目前設計為單一 consumer，未來可加入 group 概念（類似 Kafka consumer group）

4. **Dead Letter Queue**
   - 當訊息 delivery_count 超過閾值，自動移至 DLQ

5. **持久化**
   - 目前依賴 KV engine 的 flush，訊息不會丟失
   - 未來可加入定期 snapshot

6. **效能優化**
   - 目前每次 enqueue/dequeue 都會 scan entire index，未來可優化為 O(log n) 存取
   - 可考慮使用專用的 sorted set 結構

7. **監控與度量**
   - 缺少 metrics 導出（吞吐量、延遲、佇列深度等）
   - 未來可整合 Prometheus

|------|------|
| `Msgq::new(engine_type)` | 建立記憶體型 Msgq |
