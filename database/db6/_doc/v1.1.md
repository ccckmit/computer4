## 完成項目

### 布林查詢 (Boolean Queries)
- [x] AND 查詢（預設，多個 term 都需匹配）
- [x] NOT 查詢（前綴 `-` 或 `!` 排除）
- [x] search_boolean() 方法

### 前綴匹配 (Prefix Matching)
- [x] search_prefix() 方法
- [x] 支援 CJK 前綴搜尋

### BM25 排序
- [x] search_bm25() 方法
- [x] 回傳 (doc_id, score) 向量
- [x] 依分數排序

## API

```rust
use db6::{FtsIndex, MemoryEngine};

let engine = MemoryEngine::new();
let mut index = FtsIndex::new(engine);

// 布林查詢
let results = index.search_boolean("Hello -World").unwrap();

// 前綴搜尋
let results = index.search_prefix("資料").unwrap();

// BM25 排序搜尋
let results = index.search_bm25("關鍵字").unwrap();
// 回傳: Vec<(doc_id, score)>
```
