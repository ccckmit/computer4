## 完成項目

### 1. FTS API 標準化

FtsIndex 已可搭配任意引擎使用：

```rust
use db6::FtsIndex;

// FtsIndex<E> where E: StorageEngine
let mut fts = FtsIndex::new(engine);
fts.insert(doc_id, text)?;
let results = fts.search(query)?;
```

### 5. Usage

```rust
use db6::{FtsIndex, StorageEngine};

let engine = BTreeMemoryEngine::new();
let mut fts = FtsIndex::new(engine);

// 新增文件
fts.insert(1, "Hello world").unwrap();
fts.insert(2, "資料庫系統").unwrap();

// 搜尋
let results = fts.search("world").unwrap();  // 返回 Vec<u64>
let bm25 = fts.search_bm25("database").unwrap(); // 返回 Vec<(u64, f64)>
```
