## API 使用

```rust
// HashMemoryEngine 持久化
let mut engine = HashMemoryEngine::open("/data/myapp")?;
engine.put(1, b"key", b"value")?;
engine.flush()?;  // 寫入磁碟

// reopen - 資料正確讀回
let engine = HashMemoryEngine::open("/data/myapp")?;
let val = engine.get(1, b"key")?;  // 回傳 Some(value)

// BTreeMemoryEngine 持久化 (相同用法)
let mut engine = BTreeMemoryEngine::open("/data/myapp")?;
```

## 技術細節

- 使用 bincode 序列化/反序列化
- 每次 open 從磁碟讀取資料重建 in-memory 結構
- 不依賴 mmap 位址，重啟後資料正確

## 已知限制

- flush() 使用直接寫入，突發停止可能導致資料損壞
- 需要改用 temp file + atomic rename
