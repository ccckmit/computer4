2026-05-16

## 完成項目

### 1. 模組化重構

將 `lsm.rs` 重構為 `lsm/` 目錄：
- `lsm/mod.rs` - 模組入口
- `lsm/engine.rs` - LsmEngine 主體
- `lsm/memtable.rs` - MemTable
- `lsm/sstable.rs` - SSTable 磁碟儲存
- `lsm/wal.rs` - WAL Write-Ahead Log
- `lsm/bloom.rs` - Bloom Filter

### 2. 持久化支援

- [x] 實作 open(path) 讀寫磁碟
- [x] 實作 SSTable 寫入 (bincode 格式)
- [x] 實作 WAL

### 3. 效能優化

- [x] Bloom Filter 加速查詢

### 4. 架構設計

```
LsmEngine
├── path: Option<PathBuf>       # 資料目錄
├── memtable: RwLock<MemTable> # 記憶體緩衝
├── sstables: RwLock<Vec<SSTable>> # 磁碟資料
├── bloom: RwLock<BloomFilter> # 查詢加速
├── wal: RwLock<Option<Wal>>   # Write-Ahead Log
└── tx_buffer                  # 交易緩衝
```
