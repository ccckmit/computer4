# v3.3 完成項目（Transaction Fluent + 多元 WHERE）

## 日期：2026-05-16

## 新功能

### 1. Transaction Fluent Interface

```rust
let temp_dir = std::env::temp_dir().join("db6_tx");
let mut db = Db::open("lsm", &temp_dir).unwrap();

db.table("users").put(b"1", b"Alice").unwrap();

// Begin transaction, commit
db.begin_tx()
    .commit()
    .unwrap();

// Begin transaction, rollback
db.begin_tx()
    .rollback()
    .unwrap();
```

**注意**：只有 `lsm` 和 `btree` 引擎支援交易（memory 引擎不支援）。

### 2. 多元 WHERE 條件

支援 AND/OR 組合條件：

```rust
// AND 條件
let result = db.select("key, value")
    .from("users")
    .where_("key = 2 AND value = Bob")
    .execute()?;

// OR 條件
let result = db.select("key, value")
    .from("users")
    .where_("key = 1 OR value = Charlie")
    .execute()?;
```
