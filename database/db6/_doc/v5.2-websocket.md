# v5.3 WebSocket Transport

## 完成項目

- WebSocket handler 在 `/ws` 路由
- 與 HTTP/REST 共用同一 port (50052)
- 統一 request/response 格式
- 支援所有 KV 操作

## API

### WebSocket 端點

- URL: `ws://localhost:50052/ws`

### 訊息格式

請求:
```json
{
  "action": "put|get|delete|batch_put|scan|range_delete|stats|health",
  "id": 1,
  "table_id": 1,
  "key": "key",
  "value": "value",
  "start": "start_key",
  "end": "end_key",
  "pairs": [{"key": "k1", "value": "v1"}]
}
```

回應:
```json
{
  "id": 1,
  "ok": true,
  "value": "value",
  "found": true,
  "pairs": [{"key": "k1", "value": "v1"}],
  "stats": {"key_count": 10, "size_bytes": 1024, "engine": "memory"},
  "error": null
}
```

### 支援的 action

| Action | 參數 | 回應 |
|--------|------|------|
| health | - | `{ok: true}` |
| put | table_id, key, value | `{ok: true}` |
| get | table_id, key | `{value, found}` |
| delete | table_id, key | `{ok: true}` |
| batch_put | table_id, pairs | `{ok: true}` |
| scan | table_id, start, end | `{pairs: [{key, value}]}` |
| range_delete | table_id, start, end | `{ok: true}` |
| stats | - | `{key_count, size_bytes, engine}` |

## 實作

- `src/server/websocket.rs` - WebSocket handler
- `src/server/http.rs` - 路由設定

## 用戶端支援
