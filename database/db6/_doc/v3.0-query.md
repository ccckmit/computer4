### 2. API 風格

**KV Operations (Fluent):**
```rust
// Method Chaining
db.table("users")
    .put(b"key1", b"value1")?
    .put(b"key2", b"value2")?
    .flush()?;

// Simple operations
db.table("users").put(b"key", b"value")?;
db.table("users").get(b"key")?;
db.table("users").scan(b"", b"")?;
```

**SQL-like Query (Fluent):**
```rust
db.select("*")
    .from("users")
    .where("age > 18")
    .order_by("name")
    .limit(10)
    .execute()?;
```
