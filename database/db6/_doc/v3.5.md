# v3.5 完成項目（HAVING, OFFSET, Batch INSERT）

## 日期：2026-05-16

## 新功能

### 1. HAVING 子句

對 GROUP BY 結果進行過濾：

```rust
let result = db.select("COUNT(*), value")
    .from("users")
    .group_by("value")
    .having("COUNT(*) > 2")
    .execute()?;
```

### 2. OFFSET 分頁支援

```rust
// 分頁查詢
let page1 = db.select("key, value")
    .from("users")
    .order_by("key")
    .limit(3)
    .offset(0)
    .execute()?;  // rows 1-3

let page2 = db.select("key, value")
    .from("users")
    .order_by("key")
    .limit(3)
    .offset(3)
    .execute()?;  // rows 4-6

let page3 = db.select("key, value")
    .from("users")
    .order_by("key")
    .limit(3)
    .offset(6)
    .execute()?;  // rows 7-9
```

### 3. Batch INSERT 優化

InsertQuery::execute() 現在使用 batch_put() 而非個別 put()：

```rust
db.insert()
    .into_table("users")
    .values(vec![("1", "Alice"), ("2", "Bob"), ("3", "Charlie")])
    .execute()?;
```
