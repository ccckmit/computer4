Browser5

Welcome to Browser5 — a minimal web browser built with Rust + egui, powered by a custom HTML/CSS engine and the js4 JavaScript runtime.

Features

  • HTML rendering via scraper/ego-tree DOM
  • CSS styling — colors, fonts, borders, backgrounds, margins, padding, and rounded corners
  • JavaScript execution via js4 engine with data-js-var variable binding
  • Images — PNG, JPEG, GIF, and more via the image crate
  • Canvas element with dimension display

This page demonstrates the full rendering pipeline: HTML parsing, CSS selector matching, style computation, and interactive UI. Click any demo above to see JS-driven interactivity.
